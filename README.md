## 今天如何学习

1. 阅读当前专题的 `README.md`；
2. 运行 `demo.py`，核对输入和输出；
3. 按推荐顺序完成 `starter.py`；
4. 运行 `check.py`；
5. 完成至少一道变式；
6. 确实需要时再查看 `reference/solution.py`。

```powershell
python 02_machine_learning/01_linear_regression/demo.py
python 02_machine_learning/01_linear_regression/check.py
```
