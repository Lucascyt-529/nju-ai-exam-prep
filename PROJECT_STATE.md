## 总目标

建设一套用于南京大学人工智能学院预推免机试的长期训练仓库，覆盖：

1. Python 完整程序与无 OJ 自检；
2. 文件读写、NumPy、pandas 与数据处理；
3. 以周志华《机器学习》为主线的完整知识覆盖与 NumPy 核心算法手写；
4. 常见算法题、综合任务和限时模拟；
5. 能迁移、能解释、能独立实现，而不是背参考答案。

## 已确认的长期决定

- 学习者的 NumPy 和 pandas 基础较薄弱，旧学习记录不能作为当前掌握证据。
- 采用“建设一批 → 实际学习/诊断 → 调整粒度 → 建设下一批”的循环。
- 学生版、参考实现和测试分离；未经请求不替学习者填写 `starter.py`。
- 机器学习核心实现默认只依赖 Python 标准库和 NumPy。
- pandas 单独教学，用于表格数据处理，不替代 NumPy 核心算法。
- 所有完整题目必须覆盖读取、检查、处理、算法、输出和自测。
- 未核对教材版次与实际目录前，不得声称周志华《机器学习》已经无遗漏覆盖。
- 用户已允许当前仓库的小批次改动在验证后直接提交并推送到 `main`；推送前仍须检查状态和测试。
