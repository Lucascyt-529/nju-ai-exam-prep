# 模拟机试2：迁移综合卷

这是一套120分钟、150分的第二次模拟。它不复用模拟卷1的BFS和逻辑回归表面形式，重点检查能否把已经见过的基础迁移到分组汇总、带权最短路、PCA与kNN完整流水线。

## 正式计时前

1. 新建空白作答目录，只复制`student/`中的三个`.py`文件；
2. 计时中只查看`PAPER.md`、`data/`和自己的代码；
3. 不查看`reference/`、`expected/`、评分器、测试或Git历史；
4. 只确认Python与NumPy可导入，不提前运行正式数据；
5. 每题记录开始、首次运行、首次正确样例和结束时间。

## 时间分配

| 题目 | 建议时间 | 分值 | 到时保底 |
| --- | ---: | ---: | --- |
| 第1题 CSV分组 | 15分钟 | 25分 | 严格读取、分组总额和表头 |
| 第2题 Dijkstra | 30分钟 | 45分 | 先输出最短距离，再恢复路径 |
| 第3题 PCA+kNN | 75分钟 | 80分 | 先跑通无泄漏预测，再补选参、保存恢复和严格格式 |

每题至少留一个极小自测。第2题要测不可达和等距路径；第3题要主动检查`(n,d)`、`(n,)`、PCA分量形状与改变测试集不改变模型。

## 结束后

停止计时并保留原始作答，假设目录为`my_submission/`：

```text
python grade_submission.py --submission my_submission
```

自动分只做初筛。查看`SCORING.md`完成人工源码复核，再用`REVIEW_TEMPLATE.md`记录首个阻塞、形状/axis错误、泄漏、平局规则、文件格式和迁移题表现。
