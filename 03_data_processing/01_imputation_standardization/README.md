# 数据处理1：缺失值填补与标准化

## 先理解 fit 和 transform

假设训练集某特征均值为10，测试集该特征均值为100。测试时仍必须使用训练集的10进行填补或中心化，因为测试数据在训练阶段不可见。

正确流程：

```text
训练集 -> fit 得到填充值、均值、标准差
训练集 -> transform
测试集 -> 使用同一组参数 transform
```

错误流程：

```text
测试集 -> 重新计算自己的均值或标准差
```

后一种做法泄漏了测试集信息，并且让同一个原始数值在训练和测试时采用不同变换。
