# 文件读写

本模块把文件读取、格式检查和结果保存作为独立能力训练。

## 当前专题

| 专题 | 训练目标 | 状态 |
| --- | --- | --- |
| `01_text_table/` | UTF-8 文本、空行、数值表、列数校验、路径与结果文件 | 已建参考闭环，等待学生练习 |
| `02_clean_csv/` | 表头、纯数值 CSV、`loadtxt`、特征/标签切分、`savetxt` | 已建参考闭环，等待学生练习 |
| `03_missing_csv/` | 缺失值 CSV、`genfromtxt`、均值填补、严格输出格式 | 已建参考闭环，等待学生练习 |
| `04_mixed_csv/` | 标准库 `csv`、字符串字段、逐行校验与严格输出 | 已建参考闭环，等待学生练习 |

`.npy` 和 `.npz` 的参数保存与恢复将在综合数据准备任务中使用。专题可以提前建设，但只有完成相应学生版和迁移题后才记录为掌握。
