# 练习2：读取TXT数值表并保存列均值

## 题目

输入文件由若干行空白分隔的浮点数组成，允许出现空行，但所有有效行的列数必须一致。读取数值表，计算每一列的平均值，并写入指定输出文件，保留6位小数。

运行形式：

```bash
python 01_file_io/01_text_table/starter.py --input <输入文件> --output <输出文件>
```

## 必须处理

- UTF-8 文本和空行；
- 文件不存在；
- 空文件；
- 非数值字段；
- 不一致的列数；
- NaN 和无穷大；
- 输出目录不存在；
- 输出精度和末尾换行。

## 样例

参考实现运行命令：

```bash
python 01_file_io/01_text_table/reference/solution.py --input 01_file_io/01_text_table/data/sample.txt --output result.txt
```

生成的 `result.txt` 应与 `expected/column_means.txt` 完全一致。

请先独立完成 `starter.py`。`reference/solution.py` 用于完成练习后的核对，不作为第一次作答的起点。

## 分层引导

1. 先用 `Path.read_text(encoding="utf-8")` 读取并打印原始文本；
2. 使用 `splitlines()` 观察每一行，不急着转换成数组；
3. 用 `strip()` 判断空行，用 `split()` 拆分字段；
4. 逐行转成浮点数列表，并检查列数；
5. 转换为二维 NumPy 数组，写下它的 `shape`；
6. 计算均值并写入文件；
7. 最后处理文件不存在、空文件和错误格式。

每完成一步都使用当前的三行样例检查中间结果，不要一次复制完整流程。

## 迁移题

3. 将结果改成“一列一个数”的文件格式；
4. 构造包含文字字段的输入，说明程序应该在哪一步拒绝它。
