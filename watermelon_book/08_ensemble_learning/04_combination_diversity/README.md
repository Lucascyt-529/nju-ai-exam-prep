# 集成学习4：结合策略与多样性

## 常见结合策略

- 回归平均：对 `(B,n)` 个体预测按学习器轴平均；
- 加权平均：权重非负且和为1；
- 硬投票：每个分类器给标签，票数最多者获胜；
- 软投票：先平均 `(B,n,K)` 类别概率，再选最大概率类别；
- 学习法：用训练得到的次级学习器组合输出，必须用折外预测避免泄漏（本专题只讲边界，不伪装实现完整Stacking）。

平局规则必须确定。本实现硬投票选择排序后较早类别，软投票选择较早类别下标。

## 二分类成对多样性

对两个分类器相对真实标签的正确性计数：

```text
n11: 两者都对   n00: 两者都错
n10: A对B错     n01: A错B对
```

由此计算：

```text
disagreement = (n10+n01)/n
double_fault = n00/n
Q = (n11*n00-n10*n01)/(n11*n00+n10*n01)
```

Q的分母为0时指标未定义，返回NaN而不是伪造0。

## 回归误差—分歧恒等式

等权平均回归满足逐样本恒等式：

```text
ensemble_squared_error
= mean(individual_squared_error)
  - mean((individual_prediction-ensemble_prediction)^2)
```

多样性项越大可能降低集成误差，但前提是个体误差没有同时变得过大；“越不同越好”并不成立。
