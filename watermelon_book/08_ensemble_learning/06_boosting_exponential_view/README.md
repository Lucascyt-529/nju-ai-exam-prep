# 集成学习6：AdaBoost的指数风险视角

## 从加性模型开始

AdaBoost的实值得分是基学习器的线性组合：

```text
F_T(x) = sum_t alpha_t * h_t(x)
h_t(x) in {-1,+1}
```

对标签`y in {-1,+1}`，经验指数风险为：

```text
mean(exp(-y * F_T(x)))
```

若已知`P(y=+1|x)=p`，逐点最小化条件指数风险可得`F*(x)=0.5*log(p/(1-p))`，其符号与概率较大的类别一致。这解释了为何可以用连续可微的指数损失替代分类0/1损失；不表示有限数据上的AdaBoost自动恢复真实后验概率。

## 一轮为什么得到这个alpha

设当前分布下弱分类器错误率为`e`，加入`alpha*h`后的单轮因子是：

```text
Z(alpha) = exp(-alpha)*(1-e) + exp(alpha)*e
```

令导数为0：

```text
alpha* = 0.5*log((1-e)/e)
```

`e<0.5`时`alpha>0`；`e=0.5`时权重为0；`e>0.5`时为负，标准AdaBoost因此要求当前弱学习器优于随机猜测。完美错误率0对应无穷大理论权重，工程实现需提前停止或截断，不能把数值下限冒充数学等式。

## 分布更新与指数风险是同一件事

```text
D_{t+1}(i) = D_t(i) * exp(-alpha_t*y_i*h_t(x_i)) / Z_t
```

从初始分布`D_1`出发：

```text
D_{t+1}(i) proportional to D_1(i)*exp(-y_i*F_t(x_i))
exponential_risk(F_t) = product_{s=1..t} Z_s
```

因此做错或当前总间隔较小的样本会得到更高权重。参考实现同时保存得分、分布、指数风险、每轮`Z_t`和累计乘积，用数值恒等式检查推导。

训练中`1[yF<=0] <= exp(-yF)`，所以训练错误率不超过经验指数风险。这个上界下降不等于测试误差必然下降，也不消除噪声样本权重过大的风险。

## 重赋权与重采样

若基学习器支持`sample_weight`，直接按`D_t`重赋权训练。若不支持，可按`D_t`有放回重采样，再在重采样数据上普通训练。重采样只是有限随机近似：

- 固定种子可复现，但不同种子样本计数会变；
- 高权重样本更常出现，不保证每次频率精确等于概率；
- 某轮弱学习器不优于随机猜测时，重采样可提供重启机会，但不能保证成功。

## 学习任务

1. 手算`e=0.25`时的`alpha`和`Z`；
2. 对`Z(alpha)`做中心差分，验证`alpha*`附近导数为0；
3. 手算一次分布更新并检查和为1；
4. 用两轮预测矩阵验证风险等于`Z`的累计乘积；
5. 从累计得分直接重建最终分布，与逐轮更新比较；
6. 验证训练错误率不超过指数风险，并说明测试集上不能照搬此训练恒等式；
7. 对同一分布使用两个随机种子重采样，比较频数而非要求完全相同；
8. 解释完美弱分类器、随机质量弱分类器和反向弱分类器分别对应什么`alpha`。

参考闭环不等于学习者已经独立掌握。
