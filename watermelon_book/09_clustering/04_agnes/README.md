# 聚类4：AGNES凝聚层次聚类

AGNES从每个样本一个单例簇开始，反复合并距离最近的两个簇，直到达到目标簇数。

## 三种链接

给定两簇A、B及样本间距离：

```text
single:   min_{a in A,b in B} dist(a,b)
complete: max_{a in A,b in B} dist(a,b)
average:  mean_{a in A,b in B} dist(a,b)
```

单链接容易产生链式连接；全链接强调最远点；平均链接在所有跨簇样本对上平均。三者使用同一欧氏样本距离，但簇间定义不同。

## 确定性合并

初始簇编号为样本下标 `0..n-1`，新簇编号依次为 `n,n+1,...`。距离平局时选择簇编号字典序更小的一对。历史记录：左右簇编号、新簇编号、合并距离及排序后的成员。

最终标签按各簇最小成员下标排序后编号 `0..k-1`，避免内部新簇编号影响输出标签。

## 复杂度边界

教学实现每轮直接比较所有活动簇对，并从预计算样本距离矩阵取子矩阵，清晰但不是大数据高效实现。

## 学生任务

1. 手算两个簇的三种链接距离；
2. 从四个一维点完成前两次合并；
3. 解释单链接链化现象；
4. 验证平局时选择哪一对；
5. 检查标准链接下合并距离非下降；
6. 从合并历史切出指定簇数。

参考实现通过不等于学习者已经独立掌握。
