# 聚类1：距离与性能度量

## 距离是聚类算法的几何基础

数值特征的Minkowski距离：

```text
dist_p(x,z) = (sum_j |x_j-z_j|^p)^(1/p)
```

`p=1` 是曼哈顿距离，`p=2` 是欧氏距离。无序类别特征使用简单匹配/Hamming距离：不同列的比例。

不同数值特征量纲差异很大时，应先基于训练数据缩放；本专题不在距离函数中偷偷标准化。

## 内部指标与外部指标

- 内部指标只使用 `X` 和聚类结果：簇内平方和SSE、轮廓系数；
- 外部指标还需要真实分组：Rand指数、调整Rand指数（ARI）。

无真实标签时不能计算外部指标；训练类别标签也不应被聚类算法偷看。

## 轮廓系数

```text
a(i): 样本与本簇其他样本平均距离
b(i): 样本与最近其他簇平均距离
s(i) = (b-a)/max(a,b)
```

单例簇没有簇内其他样本，本实现按常见约定令轮廓值为0。

## Rand与ARI

两种划分对每一对样本判断“同簇/异簇”。Rand指数是判断一致的样本对比例。ARI再减去随机划分的期望并归一化，因此可能为负；类别编号置换不改变两者。

## 学生任务

1. 手算两点的L1/L2距离；
2. 写出一个类别向量的Hamming距离；
3. 手算两个小簇的SSE；
4. 手算一个样本的a、b和轮廓值；
5. 写出四种样本对计数；
6. 验证标签编号置换不影响Rand/ARI；
7. 解释没有真值标签时能用哪些指标。

参考实现通过不等于学习者已经独立掌握。
