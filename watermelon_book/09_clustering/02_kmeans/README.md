# 聚类2：K-means与K-means++

K-means最小化样本到所属中心的平方距离和：

```text
J = sum_i ||x_i - center[label_i]||^2
```

每轮交替两步：固定中心选最近簇；固定标签取每簇样本均值。两步都不会增加目标，因此SSE历史应当不升。

## 形状主线

```text
X:          (n,d)
centers:    (k,d)
distances:  (n,k)
labels:     (n,)
```

`argmin(axis=1)` 为每个样本选择中心；距离平局选择较早中心下标。

## K-means++

第一个中心按固定随机种子选择；后续样本以“到最近已选中心的平方距离”为权重抽样。远离已有中心的样本更可能被选中。要求 `k` 不超过不同坐标点数量，避免所有剩余权重都为0。

## 空簇

更新时某簇可能没有样本。参考实现依次选择当前到所属中心距离最大的、尚未用于重置的样本作为该空簇新中心；距离平局选择较早样本。下一次分配再决定其成员，不把空均值变成NaN。

## 收敛

若新旧标签完全相同，或最大中心移动不超过 `tolerance`，停止。历史记录每次完整“分配—更新—重新分配”后的SSE，并保留初始中心对应SSE。

1. 预测距离矩阵形状；
2. 手算一次标签分配；
3. 手算一次中心均值更新；
4. 手算K-means++第二中心权重；
5. 构造空簇并执行重置；
6. 验证SSE不增加；
7. 解释不同初始化为何可能到达不同局部最优。
