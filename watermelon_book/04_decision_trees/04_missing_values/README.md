# 决策树4：离散/连续缺失值与样本权重

## 划分时的两个问题

对某个候选属性，需要分别回答：

1. 属性值缺失时，怎样评价这个属性是否值得划分？
2. 决定划分后，缺失样本应该进入哪个分支？

本专题不先填补缺失值。每个训练样本带权重，初始通常为1。

## 带缺失值的信息增益

只使用该属性非缺失的样本计算熵和分支条件熵，再乘非缺失样本权重占总权重的比例：

```text
gain_missing = known_weight / total_weight
               * (entropy_known - weighted_child_entropy_known)
```

因此一个只在很少样本上有值的属性，即使已知部分划分完美，也会受到有效比例惩罚。

## 缺失训练样本的分支传播

已知样本只进入自己的分支，权重保持不变。缺失样本进入所有分支，分到每个分支的权重等于：

```text
sample_weight * branch_known_weight / all_known_weight
```

同一个缺失样本会在多个分支出现，但分支权重之和仍等于原权重。`branch_weight_plan` 专门把这个矩阵显示出来，方便手算和检查权重守恒。

## 连续属性联合处理

连续属性的候选阈值只由非缺失、正权重样本的不同取值中点产生。对每个阈值，先在已知样本上计算左右二分的信息增益，再乘已知权重占总权重的比例。选定阈值后，已知样本只进入左右之一，缺失样本按左右已知权重比例同时进入两个分支；连续属性在子节点仍可再次使用。

## 缺失值预测

若预测样本在当前划分属性上缺失，就沿所有子树获得类别概率，再按训练时保存的分支比例加权求和。若出现训练时未见的非缺失属性值，则回退到当前节点的带权类别分布。

## 当前边界

- 当前实现处理离散/连续数值特征和`np.nan`；
- 核心覆盖带权熵、有效比例、离散/左右分支传播、连续特征复用和概率汇总；
- 剪枝与缺失权重的联合版本尚未整合，不能用本专题替代剪枝专题；
- 多变量节点与缺失值的联合是额外扩展，不属于当前实现承诺。

## 学生任务和迁移

1. 手算已知分支权重为2和1时的分支比例；
2. 把权重为2的缺失样本分到两个分支并验证总和仍为2；
3. 手算4个已知样本完美划分、1个缺失样本时的带缺失增益；
4. 比较直接丢弃缺失样本与乘有效比例的区别；
5. 对缺失预测样本手算两个子树概率的加权和；
6. 解释为什么训练集拟合的分支比例不能用测试集重新计算。
7. 只用非缺失连续值写出候选阈值，并手算每个阈值的有效比例增益；
8. 对一个缺失样本画出左右两条训练路径，验证权重守恒；
9. 构造三个连续区间，验证同一特征可在后代节点重复使用。
