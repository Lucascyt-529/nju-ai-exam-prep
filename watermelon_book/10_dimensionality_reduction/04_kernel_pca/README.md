# 降维与度量学习 4：核 PCA

核 PCA 不显式构造高维特征，而是用核函数得到样本在特征空间中的内积矩阵，再对中心化核矩阵做特征分解。

## 训练核矩阵中心化

```text
K.shape = (n, n)
Kc = K - row_mean - column_mean + grand_mean
```

中心化后每行、每列均值都接近0。线性核 `K=X@X.T` 时，核 PCA 与普通 PCA 描述同一个中心化线性子空间。

## 谱坐标与新样本

若 `Kc v_j = lambda_j v_j`：

```text
训练坐标第j列 = sqrt(lambda_j) * v_j
新样本坐标第j列 = K_new_centered @ v_j / sqrt(lambda_j)
```

新样本核矩阵形状是 `(n_new, n_train)`。它必须使用训练核矩阵保存的列均值和总均值进行中心化，不能用测试集重新拟合训练统计量。

参考实现支持线性、多项式和RBF核。只选择严格为正的特征值，避免除以接近0的谱值。

## 学生任务

1. 手算线性核矩阵；
2. 对核矩阵完成四项中心化；
3. 检查中心化后的行列均值；
4. 从特征值和特征向量构造训练坐标；
5. 用训练样本再次调用新样本投影，核对坐标一致；
6. 比较线性核 PCA 与普通 PCA 的距离；
7. 在非线性数据上改变RBF `gamma`，观察嵌入变化。
