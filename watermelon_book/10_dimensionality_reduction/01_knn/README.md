# 降维与度量学习 1：k 近邻

kNN 没有显式训练参数。预测时，它计算查询样本与训练样本的距离，取最近的 `k` 个邻居，再投票或求平均。

## 形状先行

```text
X_train.shape = (n_train, n_features)
X_query.shape = (n_query, n_features)
distance_matrix.shape = (n_query, n_train)
neighbor_indices.shape = (n_query, k)
neighbor_distances.shape = (n_query, k)
```

距离矩阵每一行属于一个查询样本，每一列属于一个训练样本。因此“为每个查询样本找最近邻”是在 `axis=1` 的每一行内排序。

## 分类和回归

- 分类：邻居标签投票；
- 回归：邻居目标值求平均；
- `weights="uniform"`：每个邻居权重相同；
- `weights="distance"`：距离越近权重越大。

若距离加权时存在距离恰为0的邻居，只使用这些完全重合样本，避免 `1 / 0`。多个0距离分类邻居继续投票；多个0距离回归邻居取均值。

## 确定性规则

1. 距离相同的训练样本，先选下标更小者；
2. 分类总票数或总权重相同，选择排序更早的类别；
3. `k` 不能超过训练样本数。

这些规则不是所有库的唯一选择，但机试完整程序必须给平局一个可解释且可复现的定义。

## 学生任务

1. 手算两个查询点到三个训练点的距离矩阵；
2. 写出每行排序后的最近邻下标；
3. 分别完成均匀投票与距离加权投票；
4. 解释为什么单个查询点仍应保持形状 `(1, d)`；
5. 构造分类平局并验证规则；
6. 构造0距离重复样本并说明如何处理；
7. 改变 `k`，观察欠拟合与对噪声的敏感性。

先运行 `guided_demo.py`，再补全 `starter.py`。参考答案仅用于核对和纠错。
