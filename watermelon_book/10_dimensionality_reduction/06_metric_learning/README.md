# 降维与度量学习 6：成对约束度量学习

欧氏距离默认每个特征同等重要。Mahalanobis平方距离引入矩阵 `M`：

```text
d_M(x_i, x_j)^2 = (x_i - x_j).T @ M @ (x_i - x_j)
```

`M.shape == (d,d)`。为使平方距离非负，`M` 必须半正定（PSD）。若 `M=L.T@L`，这等价于先做线性变换 `Lx` 再计算欧氏距离。

## 教学目标函数

参考实现用标签生成所有样本对：同类对希望靠近，异类对希望至少达到给定平方间隔。

```text
same loss = d_M^2
different loss = max(0, margin - d_M^2)
regularization = ||M-I||_F^2
```

每步沿梯度下降后，把对称矩阵特征值裁剪到非负，投影回PSD锥；再用回溯步长保证记录的目标不增。这个小模型用于理解“学习距离”的核心，不等同于所有成熟度量学习算法。

## 学生任务

1. 手算对角 `M` 下的两点距离；
2. 解释增大某个对角元素如何放大对应特征；
3. 写出同类对和违反间隔的异类对梯度；
4. 对矩阵做对称化和PSD投影；
5. 比较学习前后的类内/类间距离；
6. 检查目标历史和最小特征值。
