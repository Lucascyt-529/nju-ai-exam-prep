# 降维与度量学习 2：经典 MDS

经典多维缩放（MDS）从样本间距离出发，寻找一组低维坐标，使低维空间中的两两距离尽量保留原距离。

## 从距离到内积

设距离矩阵 `D.shape == (n, n)`，中心化矩阵为：

```text
J = I - 11^T / n
B = -1/2 * J @ (D ** 2) @ J
```

若距离确实来自中心化的欧氏坐标 `X_centered`，则：

```text
B = X_centered @ X_centered.T
```

所以 `B` 是样本之间的内积矩阵。对 `B` 做特征分解，取最大的若干个非负特征值：

```text
B = V @ diag(lambda) @ V.T
Z = V_q @ diag(sqrt(max(lambda_q, 0)))
```

`Z.shape == (n, q)`，每一行是一个样本的新坐标。

## 为什么要平方、双中心化和开方

- `D ** 2`：距离平方可展开为两个范数和一个内积；
- 左右乘 `J`：消去只和单个样本范数有关的行列项；
- `sqrt(lambda)`：内积矩阵的特征值对应坐标轴尺度的平方。

## 数值约定

- 距离矩阵必须有限、近似对称、非负且对角线为0；
- 特征值按从大到小排列；
- 很小的负特征值通常来自浮点误差，构造坐标时截为0；
- 明显负特征值说明给定距离不完全是欧氏距离，结果中保留它们供诊断；
- 特征向量整体乘 `-1` 不改变距离，参考实现固定符号以保证演示可复现。
