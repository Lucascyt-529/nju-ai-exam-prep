# 降维与度量学习 8：近邻成分分析 NCA

NCA 把近邻分类的硬投票改成概率选择。学习线性变换 `A` 后：

```text
d_ij^2 = ||A @ (x_i - x_j)||^2
p_ij = exp(-d_ij^2) / sum_{k != i} exp(-d_ik^2),  i != j
p_ii = 0
p_i = sum_{j: y_j == y_i} p_ij
```

目标最大化所有样本留一随机近邻分类正确概率的平均值。学习后的 Mahalanobis 矩阵 `M=A.T@A` 自动半正定。

参考实现计算解析梯度并用回溯梯度上升保证记录的目标不下降。它使用方阵 `A:(d,d)`，重点呈现教材的概率近邻目标；更一般实现可以学习低秩矩形变换、加入正则或使用成熟优化器。

## 形状

```text
X:             (n,d)
A:             (d,d)
probabilities: (n,n)
metric:        (d,d)
```

概率矩阵每行和为1且对角为0。距离对称，但 `p_ij` 通常不对称，因为每行归一化分母不同。

## 学生任务

1. 手算三个样本的平方距离和逐行概率；
2. 解释为什么必须排除样本自身；
3. 计算每个样本选中同类的概率；
4. 用有限差分核对 `A` 的解析梯度；
5. 验证 `A.T@A` 半正定；
6. 构造一个噪声特征并观察其度量权重下降。
