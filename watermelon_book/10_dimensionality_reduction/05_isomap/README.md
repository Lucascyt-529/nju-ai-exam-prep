# 降维与度量学习 5：Isomap

Isomap 假设数据位于低维流形上。它不用两点间直线距离直接做MDS，而是先建立局部近邻图，用图上最短路近似沿流形行走的测地距离，再做经典MDS。

```text
X -> 欧氏距离矩阵 -> k近邻图 -> Floyd最短路 -> 经典MDS -> Z
```

## 近邻图约定

每个样本选择除自身外最近的 `k` 个点；距离平局时下标小者优先。参考实现用“并集”对称化：只要 `i` 选了 `j` 或 `j` 选了 `i`，无向边就存在。无边记为 `inf`，对角线为0。

`k` 太小可能使图不连通，此时某些测地距离为 `inf`，MDS不能继续；实现会明确报错，而不是把无穷距离悄悄换成大数。`k` 太大则近邻图接近完全图，Isomap会退化得更像普通MDS。

## 学生任务

1. 手算每行最近邻下标；
2. 完成有向邻接后再做并集对称化；
3. 用Floyd更新 `D[i,j] = min(D[i,j], D[i,k]+D[k,j])`；
4. 比较弯曲路径两端的直线距离与图上距离；
5. 把测地距离送入经典MDS；
6. 改变邻居数，观察断图和“短路边”。
