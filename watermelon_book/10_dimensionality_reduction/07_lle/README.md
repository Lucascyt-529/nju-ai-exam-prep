# 降维与度量学习 7：局部线性嵌入 LLE

LLE 假设每个样本能由其局部邻居线性重构，并希望低维坐标保持同样的重构关系。

## 两阶段

1. 为每个 `x_i` 找 `k` 个近邻，求权重 `w_ij`：最小化重构误差且权重和为1；
2. 固定权重矩阵 `W`，最小化低维坐标中的同一重构误差。

对第 `i` 个样本，局部Gram矩阵：

```text
C = (neighbors - x_i) @ (neighbors - x_i).T
w = solve(C, ones); w /= w.sum()
```

当邻居数大于特征维数或邻居共线时，`C` 可能奇异，参考实现按 `trace(C)` 加正则。全局矩阵为 `M=(I-W).T@(I-W)`；最小特征值对应常数平移，需要跳过，再取后续 `q` 个特征向量。

LLE 对整体平移和正交旋转不敏感，但邻居数过小可能断裂，过大则破坏局部假设。

## 学生任务

1. 计算每个样本的最近邻下标；
2. 手算一个局部Gram矩阵；
3. 验证每行权重和为1、非邻居权重为0；
4. 构造 `M` 并解释为什么跳过最小特征向量；
5. 检查低维坐标按列中心化；
6. 比较LLE与Isomap保留的局部结构。
