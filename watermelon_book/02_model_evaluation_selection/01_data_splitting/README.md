# 模型评估1：数据划分与可复现索引

## 为什么函数返回索引

返回索引而不是直接返回 `X_train`、`X_test`，可以让同一组划分同时作用于 `X`、`y`、样本编号和其他附加信息，避免对应关系错位。

## 方法

- 留出法：一次划分训练集和测试集；
- 分层留出：让每个类别在两部分中都有代表；
- K折交叉验证：每个样本恰好一次进入验证折；
- 分层K折：分类任务中尽量保持各折类别比例；
- 自助法：有放回抽取与原数据等量样本，未抽中的样本作为袋外样本。

固定随机种子是为了可复现，不是为了反复尝试种子直到得到最好结果。

## 运行前预测

阅读 `guided_demo.py` 后回答：

1. 10个样本、`test_size=0.2` 时测试集有几个样本；
2. 为什么训练索引和测试索引不能重叠；
3. 5折验证中每个样本作为验证样本几次；
4. 自助采样为什么可能重复同一个样本，并遗漏另一些样本。

## 学生任务和迁移

完成 `starter.py` 后：

1. 用同一种子运行两次，证明结果一致；
2. 改变种子，证明样本集合不变但划分通常变化；
3. 构造类别极不均衡数据，对比分层和非分层留出；
4. 检查每个K折验证索引合并后恰好覆盖全部样本一次；
5. 统计多次自助采样中袋外样本所占比例。
