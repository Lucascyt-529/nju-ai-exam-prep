# 模型评估2：性能度量

性能度量决定“什么叫模型更好”。同一组预测在不同度量下可能排序不同，因此不能只记公式，必须理解任务目标和错误代价。

## 回归度量

- MAE：绝对误差平均，对大误差的放大较弱；
- MSE：平方误差平均，对大误差惩罚更重；
- RMSE：MSE开平方，单位与目标变量一致；
- R²：相对于“始终预测真实值均值”基线的改进程度，可能为负。

## 二分类度量

先明确正类，再计算：

```text
TP：真实为正，预测为正
FP：真实为负，预测为正
TN：真实为负，预测为负
FN：真实为正，预测为负
```

- 查准率 precision：预测为正的样本中有多少是真的正类；
- 查全率 recall：真实正类中有多少被找出；
- F1：precision和recall的调和平均；
- 特异度 specificity：真实负类中有多少被正确排除。

零分母时，本专题统一返回0，并要求调用者理解“没有预测正类”与“查准率很好”不是一回事。

## ROC、AUC与代价

ROC使用连续分数从高到低移动阈值，横轴是假正例率FPR，纵轴是真正例率TPR。AUC通过梯形积分计算，不允许把已经离散成0/1的预测当作连续排序分数。

代价敏感错误允许假正例和假反例采用不同代价。例如漏诊代价远高于误报时，单纯准确率可能掩盖关键问题。
