# 第2章：模型评估与选择

本章先解决一个核心问题：模型在训练数据上表现好，不等于对新数据表现好。训练误差用于观察拟合过程，泛化能力必须通过未参与拟合的数据估计。

## 三条不可破坏的边界

1. 划分必须发生在任何会从数据学习参数的预处理之前；
2. 验证集和测试集不能参与填补、标准化、特征选择或模型拟合；
3. 测试集用于最终估计，不能在反复调参过程中被当作验证集使用。

## 术语区分

- 训练误差：模型在参与拟合的数据上的误差；
- 验证误差：用于选择模型和超参数；
- 测试误差：选择完成后，对最终泛化能力的一次独立估计；
- 过拟合：训练表现继续改善，但对未见数据的表现恶化；
- 欠拟合：模型连训练数据中的主要规律也没有学到。
