# 模型评估4：过拟合、学习曲线与偏差—方差

## 用多项式复杂度观察欠拟合与过拟合

对一维输入构造：

```text
degree=0: [1]
degree=1: [1, x]
degree=2: [1, x, x²]
...
```

随着次数增加，训练误差通常不会上升，因为更高次模型包含更多可调参数。但验证误差不保证持续下降：模型可能开始拟合训练噪声。

本专题输出的是可检查的数值曲线，而不是只画一张无法验证的图：

- 每个次数的训练MSE；
- 每个次数的验证MSE；
- 最小验证MSE对应的次数；
- 不同训练规模下的训练/验证MSE。

## 学习曲线

学习曲线固定模型复杂度，逐步增加训练样本数。训练样本很少时，高复杂度模型可能把训练数据拟合得很好，但验证误差很大；随着数据增加，两条曲线通常逐步接近。

本实现使用传入数据的前 `train_size` 个样本。调用前应先用固定随机种子打乱一次，不能为每个规模挑选最有利的样本。

## 偏差—方差分解

在同一个真实函数上反复抽取带噪训练集并拟合模型。对固定测试位置：

```text
平均预测与真实函数的差 → 偏差
不同训练集所得预测的波动 → 方差
观测本身不可消除的扰动 → 噪声
```

平方损失下的期望误差可写成：

```text
bias² + variance + noise variance
```

有限次模拟只是近似理论期望，因此固定随机种子并增加重复次数很重要。

## 运行前预测

阅读 `guided_demo.py` 后先预测：

1. 最高次多项式的训练误差是否会小于低次模型；
2. 验证误差最小值是否一定出现在最高次数；
3. 一阶模型和九阶模型中，哪个通常偏差更大，哪个方差更大；
4. 为什么不能用测试集选择次数。
