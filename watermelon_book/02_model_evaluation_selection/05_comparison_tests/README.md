# 模型评估5：模型比较检验

## 为什么不能只比较一个平均数

两个模型在有限样本上的分数差异可能来自抽样波动。比较检验先提出“它们没有真实差异”的原假设，再计算当前结果在原假设下是否足够极端。

本专题不把“未拒绝原假设”解释成“两模型完全相同”，也不把统计显著自动解释成实际效果足够大。

## 本专题覆盖

- 二项尾概率：已知单次错误概率时，观察到某个错误数的概率；
- 配对t统计量：同一批划分上两个模型分数之差；
- 修正重采样t统计量：考虑训练集重叠造成的方差低估；
- McNemar精确检验：两个分类器在同一测试样本上的不一致错误；
- 多数据集平均秩；
- Friedman统计量；
- Nemenyi临界差和显著差异对。

## 关于临界值

Python标准库和NumPy不提供完整的t、F、学生化极差分布函数。为了不暗中使用错误近似，本专题：

- 对二项分布和McNemar使用标准库 `math.comb` 计算精确概率；
- 对t和Friedman返回统计量，由题目提供的显著性水平表格临界值作比较；
- Nemenyi函数显式接收表格给出的 `q_alpha`。

如果考试允许SciPy，可以额外计算p值，但核心实现不依赖它。

## 运行前预测

1. 两个分类器只有 `b=1`、`c=9` 个不一致错误时，是否比 `b=5`、`c=5` 更支持性能有差异；
2. 配对比较为什么必须先按同一折对齐分数；
3. Friedman比较中，最好算法的平均秩应更小还是更大；
4. 数据集数量增加时，Nemenyi临界差如何变化。
