# 绪论 1：术语、假设空间、归纳偏好与 NFL

## 基本术语

- 数据集：样本的集合；
- 示例/样本：一个具体对象；
- 属性/特征：描述对象的变量；
- 属性值：某个样本在某个属性上的取值；
- 特征向量：一个样本的全部数值特征，通常是一行 `(d,)`；
- 标记：监督学习希望预测的结果；
- 分类：标记是离散类别；回归：标记是连续数值；
- 聚类：没有监督标记，按某种相似性形成簇；
- 学习算法：从训练数据产生模型的过程；假设：算法当前认为的输入到输出映射；
- 训练误差/经验误差：训练样本上的误差；泛化误差：新样本分布上的期望误差。

## 假设空间与版本空间

在含 `n` 个离散输入点的二分类小世界中，所有可能的二进制映射共有 `2**n` 个。观察部分输入的标记后，与所有观测一致的假设组成版本空间。只要仍有未观察点，通常就有多个一致假设。

## 归纳偏好

训练数据不能唯一决定未观察点的标记，算法必须偏好某些假设。实验用“相邻预测发生变化的次数更少”表示一种简单性偏好；这只是可计算示例，不代表简单性总是正确。

## NFL 小实验的边界

对有限输入域的所有可能目标函数等权平均，两个都会记住训练标记、但对未观察点采用不同规则的学习器，平均未见点错误率相同。这个结论依赖“所有目标函数等可能”等前提；现实任务并非随机遍历所有函数，因此算法偏好与数据分布仍然重要。

## 学生任务

1. 写出 `n=3` 的全部8个二分类目标；
2. 给两个观测点，筛选版本空间；
3. 计算每个一致假设的变化次数；
4. 改变偏好规则并观察未见点预测；
5. 穷举目标函数，核对两个学习器的平均未见点错误；
6. 解释为什么NFL不能推出“学习算法没有用”。
