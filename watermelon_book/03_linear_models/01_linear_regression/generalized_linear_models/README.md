# 线性回归扩展：联系函数与广义线性模型视角

## 核心关系

普通线性回归直接令预测均值等于线性组合：

```text
mu = eta = X @ w + b
```

更一般地，引入单调可微的联系函数 `g`：

```text
g(mu) = eta = X @ w + b
mu = inverse_link(eta)
```

这里改变的是输出条件均值 `mu` 与线性预测子 `eta` 的关系，不是把输入特征
随意做一次变换。所有形状仍保持：

```text
X:       (n,d)
weights: (d,)
eta:     (n,)
mu:      (n,)
```

## 三种联系

| 名称 | `g(mu)` | `g^{-1}(eta)` | 输出范围 |
| --- | --- | --- | --- |
| identity | `mu` | `eta` | 全体实数 |
| log | `log(mu)` | `exp(eta)` | 正实数 |
| logit | `log(mu/(1-mu))` | stable sigmoid | `(0,1)` |

教材第3.2节用对数联系说明对数线性回归，第3.3节再把同一联系函数视角用于
二分类概率。这里实现logit正逆变换是为了看清衔接；对数几率回归的最大似然训练
仍在 [逻辑回归专题](../../../../02_machine_learning/02_logistic_regression/README.md)，不用“对二进制标签做logit后最小二乘”代替。

## 对数线性回归

若目标 `y` 全部严格大于0，可以拟合：

```text
log(y) = X @ w + b
prediction = exp(X @ w + b)
```

参考实现对 `log(y)` 使用 `np.linalg.lstsq`。因此它最小化的是对数尺度上的平方
误差，不是原始 `y` 尺度的平方误差；存在随机噪声时，直接 `exp` 回原尺度还可能
产生反变换偏差。本专题用透明的确定性数据理解联系函数，不宣称实现了包含响应
分布、方差函数和极大似然估计的全部统计GLM。

## 数值边界

- log联系只接受严格正数；
- logit联系只接受严格位于 `(0,1)` 的数，二分类标签0/1本身不能直接取logit；
- logit逆联系分正负区间计算，避免大绝对值指数溢出；
- log逆联系若 `exp(eta)` 溢出会明确报错；
- 所有目标和预测保持一维，拒绝 `(n,1)` 与 `(n,)` 的广播混用。
