# 线性模型3：线性判别分析（LDA）

## 几何目标

LDA寻找一个方向 `w`，把样本投影到一条直线上：

```text
projected = X @ w
```

希望同一类投影尽量集中，不同类投影均值尽量分开。二分类方向可写为：

```text
w = pinv(Sw) @ (mu1 - mu0)
```

其中 `Sw` 是两类内部散度之和。参考实现允许加入 `regularization * I`，并使用伪逆处理奇异矩阵，而不是直接求普通逆矩阵。

## 阈值

本专题使用两个投影类均值的中点作为阈值：

```text
threshold = (mu0 @ w + mu1 @ w) / 2
```

当方向按 `mu1-mu0` 定义时，投影大于等于阈值预测为1。

## Fisher准则

```text
J(w) = 类间投影均值差² / 类内投影离差平方和
```

把 `w` 同时乘任意非零常数不会改变准则值；阈值同步缩放时也不改变分类结果。

## 多分类LDA

对 `K>=3` 个类别，先计算全局均值、每类均值以及：

```text
Sw = 各类别内部散度之和
Sb = sum_k n_k * (mu_k - mu) @ (mu_k - mu).T
St = 所有样本相对全局均值的总散度
St = Sw + Sb
```

多分类投影矩阵 `W:(d,r)` 满足广义特征值关系：

```text
Sb @ W = Sw @ W @ Lambda
r <= min(d, K-1)
```

参考实现不直接对可能非对称的 `pinv(Sw) @ Sb` 做普通特征分解，而先对白化后的
对称类间散度使用 `np.linalg.eigh`。`regularization * I` 用于奇异/近奇异的类内
散度。若类别均值实际只张成更低维子空间，默认返回真实可用方向数；显式请求超过
有效秩的方向会报错，不用零特征值方向凑数。

投影后使用最近投影类中心进行透明分类。距离相同时，`np.argmin` 选择升序类别表
中更靠前的原标签。这个简单预测器用于验证监督降维表示，不取代带先验和共享协方差
概率假设的完整生成式判别规则。

## 数值和边界

- 二分类入口的`X:(n,d)`、`y:(n,)`标签固定为0/1；多分类入口接受至少三个有限数值原标签；
- 两类都必须至少有一个样本；
- 两类均值完全相同时没有可用的均值差方向，本专题明确拒绝；
- 特征重复会让 `Sw` 奇异，伪逆仍可给出有效投影；
- 所有样本整体平移会改变类均值和阈值，但不应改变预测。

1. 完成 `starter.py` 并手算二维小样例；
2. 增加一列重复特征，验证伪逆处理；
3. 所有样本加同一个偏移向量，比较预测；
4. 把权重和阈值同时乘10，比较预测；
5. 改变正则强度，比较方向和Fisher准则。
6. 对三类二维小数据手算全局均值、三个类均值及`St=Sw+Sb`；
7. 预测多分类投影矩阵的最大列数，并解释为什么不超过`K-1`；
8. 增加重复特征制造奇异`Sw`，比较零正则和正正则；
9. 将所有样本平移同一向量，验证散度矩阵不变、投影预测不变；
10. 构造三个类中心共线的数据，解释为何只有一个有效判别方向；
11. 比较原空间最近中心与LDA投影空间最近中心，说明监督投影使用了哪些标签信息。
