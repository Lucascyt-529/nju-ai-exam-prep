# 神经网络6：RBF网络与线性输出层

## 从距离变成隐藏层响应

径向基函数只关心样本到中心的距离。本专题使用高斯基函数：

```text
phi_j(x) = exp(-||x-center_j||^2 / (2 * width^2))
```

样本越接近中心，响应越接近1；距离越远，响应越接近0。

## 形状

设有 `n` 个样本、`d` 个特征和 `m` 个中心：

```text
X:                 (n,d)
centers:           (m,d)
squared_distances: (n,m)
hidden_response:   (n,m)
design_matrix:     (n,m+1)  # 最后一列是偏置1
output_weights:    (m+1,)
prediction:        (n,)
```

这里回归目标 `y` 使用 `(n,)`，因为输出层最终产生一维预测 `(n,)`。上一批网络保留输出神经元维得到 `(n,1)`，本批用向量得到 `(n,)`；两种表示都可以，关键是两边形状必须一致，不能依赖意外广播。

## 宽度的作用

- `width` 小：每个中心只在很近范围内有明显响应，模型更局部；
- `width` 大：响应更平缓，相邻中心影响范围重叠更多。

宽度必须是正有限数值。它不是“越小越好”，也不是“越大越好”。

## 输出层拟合

中心和宽度给定后，隐藏层设计矩阵已经固定，输出层就是线性回归：

```text
prediction = design_matrix @ output_weights
```

无正则时使用 `np.linalg.lstsq`，即使设计矩阵列相关也能给出最小二乘解。加入L2时只惩罚径向基权重，不惩罚最后的偏置项。

## 当前边界

本专题显式接收中心，不在内部偷偷调用K-means。中心选择和宽度调参是独立问题，将在聚类与模型选择专题中连接。当前实现是一输出回归RBF网络，不代表其他常见网络已经覆盖。

## 学生任务

1. 手算两个样本到两个中心的平方距离矩阵；
2. 比较同一距离在小/大宽度下的响应；
3. 给设计矩阵补上偏置列；
4. 用中心等于训练样本的小数据检查近似插值；
5. 制造重复中心，确认最小二乘仍返回有限结果；
6. 解释为什么本题 `y:(n,)` 是合理的，以及若改成 `(n,1)` 还要同步改哪些形状。
