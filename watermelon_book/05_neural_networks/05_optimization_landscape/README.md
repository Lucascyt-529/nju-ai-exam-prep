# 神经网络5：全局最小、局部极小与初始化

## 一个能完整分析的非凸函数

神经网络损失面维数很高，不适合刚开始时直接画全貌。本专题先研究一维双井函数：

```text
f(x) = (x^2 - 1)^2 + 0.2x
f'(x) = 4x^3 - 4x + 0.2
f''(x) = 12x^2 - 4
```

四次项保证 `x` 趋向正负无穷时 `f(x)` 都趋向正无穷。求出全部三个实驻点后，就能严格比较两个极小点谁是全局最小，而不是只在有限网格里猜测。

## 驻点不等于最小值

先检查 `f'(x)=0`，再看二阶导数：

```text
f''(x) > 0  -> 局部极小
f''(x) < 0  -> 局部极大
f''(x) = 0  -> 仅靠二阶导数仍不能判断
```

本函数有两个局部极小和中间一个局部极大。左侧井更深，因此左侧极小同时是全局最小；右侧极小只是局部极小。

## 初值决定吸引域

固定学习率做梯度下降：

```text
x = x - learning_rate * f'(x)
```

从 `x=-2` 或 `x=-0.2` 开始会进入左侧深井；从 `x=0.2` 或 `x=2` 开始会进入右侧浅井。梯度都能接近0，但终点函数值不同。

这说明：

- 梯度很小只表示接近驻点；
- 最终损失比初始低，不证明到达全局最小；
- 不同初始化可能到达不同结果；
- 多次随机初始化是经验策略，不是全局最优证明。

## 与神经网络的类比边界

这个一维例子用于理解概念，不等于真实神经网络损失面只有两个井，也不证明神经网络中的每个驻点都可用一维二阶导数简单分类。高维情形还可能出现鞍点和平坦区域。

## 学生任务

1. 手算 `f'(x)` 和 `f''(x)`；
2. 根据三个驻点的二阶导数先分类，再看函数值；
3. 预测四个初值会进入哪个井；
4. 检查固定学习率下每一步函数值是否下降；
5. 把学习率调大，观察“梯度公式正确”为什么仍可能发散；
6. 解释局部极小、全局最小和鞍点的差别。
