# 神经网络7：SOM竞争学习与拓扑邻域

## SOM没有标签

自组织映射（SOM）是无监督竞争学习。输入只有：

```text
X: (n,d)
prototypes: (k,d)
```

每个神经元保存一个 `d` 维原型，同时位于一个固定的拓扑网格。本专题使用最容易手算的一维网格，神经元位置是 `0,1,...,k-1`。

## 最佳匹配单元

对一个样本 `x:(d,)`，计算它到所有原型的平方距离：

```text
distances[j] = sum((x - prototypes[j])^2)   -> (k,)
winner = argmin(distances)
```

距离平局时 `np.argmin` 返回最早下标，使结果确定。最佳匹配单元（BMU）不是类别标签，只是当前最接近样本的原型编号。

## 邻域更新

不仅更新获胜原型，还按一维网格距离更新邻居：

```text
h_j = exp(-(j-winner)^2 / (2*radius^2))       (k,)
P_j = P_j + learning_rate * h_j * (x-P_j)    (d,)
```

获胜神经元的邻域权重是1，越远越小。代码中使用 `h[:,None]` 把 `(k,)` 变成 `(k,1)`，再与 `(k,d)` 正确广播。

## 初始化和衰减

原型在每个特征的训练最小值与最大值之间随机初始化。固定随机种子和样本顺序保证可复现。

学习率和邻域半径从初始值指数衰减到最终值：

- 训练早期：较大邻域帮助形成整体拓扑；
- 训练后期：较小邻域进行局部微调。

## 量化误差

量化误差是每个样本到最近原型的欧氏距离平均值。它能衡量表示误差，但单独降低量化误差不等于拓扑关系一定完美，也不等于完成了监督分类。

## 当前边界

本专题只实现一维神经元网格、固定样本顺序和Gaussian邻域。二维网格、随机打乱、拓扑误差与更复杂调度可作为迁移题。SOM闭环也不代表其他常见网络已经覆盖。
