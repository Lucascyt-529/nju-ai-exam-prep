# 神经网络8：ART1选择、警戒与共振

## 稳定性与可塑性

增量学习希望同时做到：

- 可塑性：新模式到来时能学习新类别；
- 稳定性：学习新模式时不过度破坏已有类别。

ART通过“候选选择 + 警戒检验”决定复用旧类别还是创建新类别。本专题实现用于二值模式的简化ART1快速学习过程。

## 输入和原型

```text
X:          (n,d)，元素只能是0或1，每行至少有一个1
prototypes: (k,d)，每个类别一个二值交集原型
```

若模式 `I` 分配给类别 `j`，快速学习更新是：

```text
w_j = I AND w_j
```

原型只会从1变0，不会把已经失去的共同特征重新变回1。

## 两个不同步骤

### 1. 选择函数排序

```text
choice_j = |I AND w_j| / (alpha + |w_j|)
```

`alpha > 0` 避免分母为0。先按选择分数降序检查类别，平局时先检查较早下标。

### 2. 警戒检验

```text
match_j = |I AND w_j| / |I|
match_j >= vigilance  -> 共振，允许更新该类别
```

选择分数高不代表一定通过警戒阈值。候选失败后继续检查下一个类别；全部失败才创建新类别。

## 警戒参数与粒度

`vigilance` 位于0到1：

- 较低：允许差异更大的模式共用类别，类别较粗；
- 较高：要求更高重合，通常产生更多、更细的类别。

“通常”不等于类别数对所有输入顺序都严格单调；ART是在线增量过程，输入顺序可能影响最终原型。

## 类别上限

若没有旧类别通过警戒检验且已达到 `max_categories`，训练明确抛出错误，不悄悄把模式塞进不合格类别。预测阶段遇到无共振类别则返回 `-1`。

## 当前边界

这是透明的二值ART1代表实现，采用快速交集学习，没有实现连续值模糊ART、补码编码或完整生物神经解释。它用于理解选择、重置、警戒和稳定性—可塑性权衡。

## 学生任务

1. 手算一个模式对两个原型的选择分数；
2. 按分数排序后逐个做警戒检验；
3. 用按位交集更新通过共振的原型；
4. 比较低/高警戒值的类别数；
5. 改变输入顺序，观察在线学习结果；
6. 解释为什么预测无共振时返回 `-1`，而不是强行选最高分。
