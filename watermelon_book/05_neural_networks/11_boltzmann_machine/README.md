# 神经网络 11：Boltzmann 机小型精确实验

Boltzmann 机为二值状态 `s in {0,1}^d` 定义能量：

```text
E(s) = -1/2 * s.T @ W @ s - b.T @ s
p(s) = exp(-E(s)/T) / Z
```

`W` 必须对称且对角线为0；`1/2` 避免一条无向连接计算两次。温度 `T>0`，低能量状态概率更高；温度升高时分布趋于平坦。

## 条件激活与 Gibbs 采样

固定其他单元时，第 `i` 个单元取1的概率为：

```text
sigmoid((b[i] + sum_j W[i,j] * s[j]) / T)
```

Gibbs 采样依次按该条件概率更新每个单元。参考实现固定更新顺序并接收随机种子，使轨迹可复现。

## 为什么先做精确小模型

`d` 个二值单元共有 `2**d` 个状态，穷举只适合很小的网络，但能精确检查概率归一化、条件概率和期望。较大网络才需要MCMC近似；本专题不把短链样本冒充精确分布，也暂不伪装成受限Boltzmann机完整训练。
