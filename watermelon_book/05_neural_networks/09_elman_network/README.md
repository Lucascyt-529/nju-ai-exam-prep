# 神经网络9：Elman上下文状态与序列前向传播

## 为什么需要状态

普通前馈网络对每个样本独立计算；Elman网络把上一时刻隐状态送回当前时刻：

```text
h_t = tanh(x_t @ Wx + h_(t-1) @ Wh + bh)
y_t = h_t @ Wy + by
```

因此，即使当前 `x_t` 相同，只要之前的输入历史不同，`h_(t-1)` 不同，当前输出也可能不同。

## 形状

设序列长度为 `T`，输入维数 `d`，隐状态维数 `h`，输出维数 `o`：

```text
X:       (T,d)
Wx:      (d,h)
Wh:      (h,h)
bh:      (h,)
Wy:      (h,o)
by:      (o,)
h_t:     (h,)
states:  (T+1,h)  # 第0行是初始状态h_0
outputs: (T,o)
```

`states` 比输入多一行，便于明确看到初始状态和每一步更新。默认初始状态是全零，也可以传入上一段序列的最终状态。

## 分段处理与状态重置

若先处理序列前半段，再把它的最终状态作为后半段的 `initial_state`，拼接后的输出应与一次处理完整序列完全一致。

如果后半段重新使用默认零状态，则叫作状态重置，结果通常不同。这是跨批次序列处理时必须明确的选择。

## 输出层

本专题使用线性输出层，专注于Elman的上下文连接。分类任务可在输出上接sigmoid/softmax，回归任务可保留线性输出；输出激活和损失取决于具体任务。

## 当前边界

这一批只有参数初始化、单步递推、序列前向传播、状态续接和序列MSE。它没有实现时间反向传播（BPTT）、截断BPTT、梯度裁剪或完整训练，因此不能把固定参数的记忆实验当成已训练循环网络。

## 学生任务

1. 写出单个时刻两个矩阵乘法的形状；
2. 手算长度2序列的 `h_1、h_2`；
3. 构造“最后输入相同、前一输入相反”的两条序列；
4. 比较整段前向与分段续接；
5. 再重置后半段状态并解释差异；
6. 说明为什么 `states` 是 `(T+1,h)` 而不是 `(T,h)`。
