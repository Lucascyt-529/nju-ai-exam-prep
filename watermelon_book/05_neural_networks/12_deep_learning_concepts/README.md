# 神经网络 12：深度学习概念实验

教材第5.6节强调多隐层带来的表示能力与训练困难，并介绍两条经典思路：

- 无监督逐层预训练：一次训练一层，再进行整体微调；
- 权共享：多个位置使用同一组连接权，卷积网络是典型例子。

## 逐层预训练实验

参考实现用线性PCA自编码器作为透明的单层预训练器：每层只看上一层编码，保存训练均值和主方向，再把编码交给下一层。

```text
X:(n,d0) -> code1:(n,d1) -> code2:(n,d2)
```

每层都能独立重构自己的输入并记录MSE。之后可以在最终编码上拟合监督线性头，展示“先用无标签数据得到表示，再用标签适配任务”的流程。真正的整体微调会继续更新所有层，本实验没有把“只训练输出头”错误称为完整微调。

## 权共享实验

长度 `L`、核宽 `K`、滤波器数 `F` 的有效一维卷积：

```text
shared parameters = K * F
locally connected parameters = (L-K+1) * K * F
```

卷积在每个位置复用同一核；局部连接层则为每个位置单独保存权重。共享减少参数并带来平移结构偏好。

## 学生任务

1. 写出两层编码的全部形状；
2. 验证每层只用自己的训练均值；
3. 手算一个一维卷积窗口；
4. 比较共享与不共享参数量；
5. 平移局部模式并观察卷积响应位置同步移动；
6. 解释预训练、监督头训练和整体微调的区别。
