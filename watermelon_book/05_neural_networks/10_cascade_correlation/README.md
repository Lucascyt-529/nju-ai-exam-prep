# 神经网络 10：级联相关结构实验

级联相关网络不一开始固定全部隐藏层，而是从简单输出模型开始，逐次加入能解释当前残差的新隐藏单元。

## 教学流程

1. 用当前特征拟合输出层并计算残差；
2. 生成若干候选隐藏单元；
3. 计算每个候选激活与残差的绝对相关系数；
4. 选择相关性最大的候选；
5. 冻结它的输入权重，把激活追加为新特征；
6. 重新拟合输出层，再进入下一轮。

第 `h` 个隐藏单元接收原始 `d` 个特征以及此前 `h` 个隐藏激活，因此其输入权重形状是 `(d+h,)`。输出层最终接收 `(d+n_hidden,)` 个特征。

参考实现用随机候选池和 `tanh` 激活展示“选择—冻结—增长”的结构，输出层使用最小二乘或岭回归。成熟级联相关算法还会专门优化候选单元的输入权重；本实验不把随机候选近似伪装成完整工程实现。

## 学生任务

1. 手算激活与残差的相关系数；
2. 解释为什么残差为常数时相关性记0；
3. 写出加入一个隐藏单元前后的特征形状；
4. 验证已加入单元在后续步骤中权重不变；
5. 从保存的隐藏单元顺序重建预测；
6. 检查训练MSE历史是否不增加。
