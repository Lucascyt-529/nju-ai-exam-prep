# 神经网络2：单隐层前向传播与标签形状

## 参数和中间结果

设输入有 `d` 个特征，隐层有 `h` 个神经元，输出层有1个神经元：

```text
X:   (n, d)
W1:  (d, h)
b1:  (h,)
z1:  (n, h) = X @ W1 + b1
a1:  (n, h) = sigmoid(z1)
W2:  (h, 1)
b2:  (1,)
z2:  (n, 1) = a1 @ W2 + b2
p:   (n, 1) = sigmoid(z2)
```

`b1:(h,)` 会沿样本方向广播到每一行，结果仍是 `(n,h)`；`b2:(1,)` 会加到每个样本唯一的输出神经元上。

## 为什么这里把标签变成矩阵

普通二分类标签常保存为 `y:(n,)`，因为每个样本只有一个标量标签。但本网络的输出刻意保留“输出神经元”这一维，所以 `p` 是 `(n,1)`。

如果直接相减：

```text
(n,1) - (n,) -> (n,n)   # NumPy广播成功，但语义错误
```

必须显式转换：

```python
y_column = y.reshape(-1, 1)  # (n,1)
error = probabilities - y_column  # (n,1)
```

因此，“一层中括号就是数组、两层中括号就是矩阵”只能当作创建数据时的初步直觉；真正可靠的判断始终是 `ndim` 和 `shape`。本专题的 `as_column_labels` 专门校验 `(n,)` 后再生成 `(n,1)`。

## 数值稳定

直接计算 `1 / (1 + exp(-z))` 时，极大负数会在 `exp(-z)` 处溢出。稳定sigmoid按正负分支计算。

二元交叉熵也不先对概率取对数，而直接从logit计算：

```text
max(z, 0) - z*y + log(1 + exp(-abs(z)))
```

这样 `z=1000` 或 `z=-1000` 仍会得到有限结果。
