# 神经网络 13：受限 Boltzmann 机与 CD-1

RBM 把单元分为显层 `v` 和隐层 `h`，同层内无连接。给定一层后，另一层各单元条件独立：

```text
P(h_j=1 | v) = sigmoid(v @ W[:,j] + hidden_bias[j])
P(v_i=1 | h) = sigmoid(h @ W[i,:] + visible_bias[i])
```

## CD-1

对一个训练批次：

1. 由数据 `v0` 计算并采样 `h0`；
2. 由 `h0` 重构并采样 `v1`；
3. 由 `v1` 计算并采样 `h1`；
4. 用正相与负相统计量之差更新参数。

```text
gradient_W = v0.T @ h0 / n - v1.T @ h1 / n
```

CD-1只运行一步短链，不是精确最大似然梯度。它速度快但有偏；不能要求每个随机批次重构误差单调下降。参考实现固定种子以便复现采样轨迹，并同时返回采样状态和梯度，便于手算核对。
