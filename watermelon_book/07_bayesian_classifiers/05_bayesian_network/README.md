# 贝叶斯分类器5：二值贝叶斯网、学习与推断

贝叶斯网用有向无环图（DAG）描述变量依赖，并把联合分布分解为：

```text
P(x_1,...,x_d) = product_i P(x_i | parents(x_i))
```

本专题只处理二值变量0/1，使每个条件概率表（CPT）都能直接检查。

## CPT轴顺序

每个变量保存一个有序父节点元组。`P(X=1|parents)` 数组的轴严格对应这个顺序。例如父节点为 `(A,B)`，则：

```text
cpt[a,b] = P(X=1 | A=a, B=b)
```

交换父节点顺序却不转置CPT会得到能运行但语义错误的模型，因此构建网络时必须校验形状。

## 为什么图必须无环

拓扑顺序要求每个父节点先于子节点。若存在环，就无法形成联合分布的有向生成顺序。参考实现使用Kahn算法，并在剩余节点无法入队时明确报错。

## 条件独立与道德图知识卡

- 顺序结构 `A -> B -> C`：给定中间结点B后，A与C条件独立；
- 同父结构 `A <- B -> C`：给定共同父结点B后，A与C条件独立；
- V型结构 `A -> B <- C`：没有观测碰撞结点B时，A与C可边际独立；观测B或B的后代后反而可能相关，这叫“解释消除”；
- 检查有向分离时，先把相关祖先子图中的V型结构两父结点相连，再把有向边改为无向边，得到道德图；删除条件变量后若两组变量被分开，则得到相应条件独立关系。

这些规则用于理解网络结构表达了哪些独立性，不把相关性直接解释成因果关系。

## 枚举精确推断

查询 `P(Q|evidence)` 时：

1. 固定证据与 `Q=0`，枚举其他隐藏变量并求联合概率和；
2. 固定证据与 `Q=1`，重复求和；
3. 将两个结果归一化。

复杂度随隐藏变量指数增长，因此这里只用于小网络、手算验证和理解推断定义；大网络需要变量消元、信念传播或采样。

## 已知结构下学习CPT

训练数据按 `variable_names` 对齐到矩阵列。对每个结点和每种父结点取值组合，分别统计当前结点为0/1的次数。`alpha=0` 是极大似然计数，正的 `alpha` 提供Laplace平滑；从未出现的父配置在无平滑时暂取0.5，避免用不存在的数据伪造偏向。

## BIC与受约束结构搜索

结构评分使用需要最小化的：

```text
-log_likelihood + 0.5 * parameter_count * log(n)
```

似然偏爱复杂结构，第二项惩罚更多父结点带来的CPT参数。本专题从空图开始，只允许沿给定拓扑顺序向前加边，每次选择令BIC下降最多的一条边。这是透明的教学版贪心搜索：它保证无环，但不保证找到所有DAG中的全局最优结构。

## Gibbs近似推断

固定证据变量后，每次只重采样一个非证据变量。其条件概率只需本结点因子和直接子结点因子，也就是Markov毯中的相关部分。经过丢弃的预热轮次后，用查询变量取1的频率估计后验。

Gibbs结果带随机误差且相邻样本相关，因此测试只要求在固定样本数下接近小网络的精确枚举结果，不把逐次估计写成必然单调收敛。

## 学生任务

1. 判断给定父节点字典是否有环；
2. 写出一个合法拓扑序；
3. 根据父节点顺序读取CPT元素；
4. 手算一个完整赋值的联合概率；
5. 枚举四节点网络的全部16个赋值并验证概率和为1；
6. 手算一个带证据的二值后验；
7. 解释为什么精确枚举不适合很多隐藏变量。
8. 在给定DAG下手算一个二值CPT；
9. 比较空图和单边图的似然、参数量与BIC；
10. 解释为什么“只沿固定顺序加边”不会产生环；
11. 手算一个结点的Markov毯条件概率；
12. 比较Gibbs估计与枚举精确后验，并改变采样数观察误差。
13. 分别判断顺序、同父和V型结构在给定/不给定中间结点时的独立性。
