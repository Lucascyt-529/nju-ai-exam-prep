# 贝叶斯分类器2：极大似然参数估计

极大似然估计（MLE）把观测数据视为固定，选择使这些数据出现概率最大的参数：

```text
theta_hat = argmax_theta product_i P(x_i|theta)
```

实际计算常改为对数似然，将乘积变成求和，避免许多小概率相乘下溢。

## 本专题的四类估计

1. 类别先验：`count(class)/n`；
2. Bernoulli参数：二值样本中1的比例；
3. 类别分布：每个取值的频率，可选Laplace平滑；
4. 高斯分布：均值、MLE方差和多元协方差。

## MLE方差为什么除以n

对高斯似然直接求导得到：

```text
variance_MLE = sum((x_i-mean)^2) / n
```

NumPy对应 `np.var(x, ddof=0)`。统计学中常见的无偏样本方差使用 `n-1`，对应 `ddof=1`；两者解决的目标不同，不能看到“方差”就默认同一个分母。

多元高斯协方差同样使用：

```text
cov_MLE = centered.T @ centered / n
```

小样本或重复特征可能产生奇异协方差。这不是实现错误；后续高斯分类器会显式加入方差下限或协方差正则化。

## MLE、似然与后验不要混淆

- 本专题根据带标签训练数据估计 `P(c)` 与 `P(x|c)` 的参数；
- 上一专题再用贝叶斯公式得到 `P(c|x)`；
- MLE本身没有给参数放先验，若对参数也放先验并最大化后验，那是MAP估计。
