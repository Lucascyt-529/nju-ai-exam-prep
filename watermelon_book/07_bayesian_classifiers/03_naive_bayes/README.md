# 贝叶斯分类器3：离散与高斯朴素贝叶斯

## “朴素”假设

朴素贝叶斯假设：给定类别后，各特征条件独立。

```text
P(c|x) ∝ P(c) * product_j P(x_j|c)
```

这个假设通常并不完全真实，但能把高维联合分布拆成许多一维条件分布，使参数估计简单、样本需求较小。

## 为什么必须使用对数概率

许多小概率相乘容易下溢为0。计算时改为：

```text
log joint(c,x) = log P(c) + sum_j log P(x_j|c)
```

预测类别只需取最大联合对数分数；若要输出后验概率，再使用减去行最大值的稳定softmax归一化。

## 离散朴素贝叶斯

每个特征分别记录训练集中出现的类别值。参考实现使用Laplace平滑，并额外保留一个“未知值桶”：

```text
P(value|class) = (count + alpha) / (class_count + alpha*(V+1))
```

因此测试集出现训练期未见取值时不会让整类概率直接变成0。不同特征的值域必须分别保存，不能把所有列的类别混在一起。

## 高斯朴素贝叶斯

连续特征在每个类别内分别估计均值与方差：

```text
means:     (K,d)
variances: (K,d)
```

常数特征会得到零方差，直接放入高斯密度会除零。参考实现使用明确的 `variance_floor`，而不是默默产生无穷或NaN。

## 形状主线

```text
X:                  (n,d)
y:                  (n,)
joint_log_scores:   (n,K)
posterior:          (n,K)
prediction:         (n,)
```

## 学生任务

1. 手算一个二特征二分类离散模型的计数和平滑概率；
2. 解释未知值桶为什么也要占一个平滑类别；
3. 将概率连乘改写成对数求和；
4. 手算一行联合对数分数；
5. 验证后验每行和为1；
6. 手算每类每特征均值和MLE方差；
7. 构造常数特征并解释方差下限；
8. 比较离散与高斯版本的适用数据类型。

参考实现通过不等于学习者已经独立掌握。
