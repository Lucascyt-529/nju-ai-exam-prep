# 贝叶斯分类器4：AODE半朴素贝叶斯

朴素贝叶斯假设所有特征在给定类别后彼此独立。AODE（Averaged One-Dependence Estimators）放宽为：每次选择一个特征作为“超父”，其余特征允许同时依赖类别与该父特征。

对父特征 `x_i`，一个SPODE分类器计算：

```text
P(c, x_i) * product_{j != i} P(x_j | c, x_i)
```

AODE再对所有训练频数不低于阈值的父特征分类器求平均。实现使用对数空间的 `log-sum-exp` 汇总，避免概率连乘下溢。

## 为什么需要父特征频数阈值

若某个父取值在训练集中只出现一次，条件统计极不可靠。`min_parent_count` 要求查询样本的该父取值在训练集中至少出现指定次数。没有任何父特征合格时，参考实现回退到同一训练集上的Laplace朴素贝叶斯，而不是返回空结果。

## 平滑与未知值

- `P(c,x_i)` 对类别、父取值组合做Laplace平滑；
- `P(x_j|c,x_i)` 为每个子特征保留未知值桶；
- 查询期未见父取值不会成为合格超父，但仍可作为其他超父下的未知子取值参与计算。

## 形状与复杂度

```text
X_train: (n,d)   query: (m,d)   scores: (m,K)
```

本实现为教学清晰度直接读取训练计数，预测复杂度高于预计算表版本，适合小型机试样例与手算核验。

## 学生任务

1. 手算一个父特征的 `P(c,x_i)`；
2. 手算一个子特征的 `P(x_j|c,x_i)`；
3. 判断给定查询有哪些合格父特征；
4. 用log-sum-exp合并两个SPODE分数；
5. 提高频数阈值，观察何时回退朴素贝叶斯；
6. 解释未知父值与未知子值的处理为何不同；
7. 构造两个高度相关特征，比较AODE与朴素分数。

参考实现通过不等于学习者已经独立掌握。
