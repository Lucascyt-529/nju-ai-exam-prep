# 贝叶斯分类器1：最小风险决策

## 分类不只是选最大概率

若后验概率为 `P(c_j|x)`，采取动作 `a_i` 的条件风险是：

```text
R(a_i|x) = sum_j loss[i,j] * P(c_j|x)
```

其中 `loss[i,j]` 表示真实类别为 `j` 时却采取动作 `i` 的代价。贝叶斯最优决策选择条件风险最小的动作。

## 0/1损失为什么等价于最大后验

0/1损失规定分类正确代价0、错误代价1：

```text
R(a_i|x) = 1 - P(c_i|x)
```

所以最小化风险正好等价于最大化后验概率。但代价不对称时，这个等价关系不再成立。

例如二分类中，假阴性代价为5、假阳性代价为1时，预测正类的阈值是：

```text
P(positive|x) >= 1 / (1+5) = 1/6
```

即使正类后验只有0.3，最大后验会选负类，最小风险决策仍会选正类。

## 从先验和似然到后验

给定类先验 `P(c)` 与类条件似然 `P(x|c)`：

```text
joint_score = P(x|c) * P(c)
posterior = joint_score / sum_c joint_score
```

分母是证据 `P(x)`。分类比较时它对所有类别相同，有时可以省略；若需要输出真正后验概率，就必须归一化。

## 形状约定

```text
priors:               (K,)
class_likelihoods:    (n,K)
posteriors:           (n,K)
loss_matrix:          (A,K)，本专题要求A=K
conditional_risks:    (n,A)
decisions:            (n,)
```

代价矩阵的“行动在行、真实类别在列”必须固定，不能用到一半再转置。

## 学生任务

1. 手算一次先验乘似然后归一化；
2. 为三分类写出0/1代价矩阵；
3. 手算每个动作的条件风险；
4. 证明0/1损失下最小风险等价于最大后验；
5. 构造一个最大后验与最小风险结论不同的例子；
6. 解释代价矩阵行列含义；
7. 比较逐样本最小风险与整批平均风险。
