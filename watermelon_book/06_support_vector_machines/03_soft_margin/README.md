# 支持向量机3：软间隔、hinge损失与参数C

## 这一专题解决什么问题

现实数据常常不能被一条直线完全分开。软间隔SVM允许部分样本进入间隔，甚至被误分类，同时仍然希望超平面尽量平坦：

```text
minimize  0.5 * ||w||^2 + C * sum(xi_i)

subject to  y_i * (w^T x_i + b) >= 1 - xi_i
            xi_i >= 0
```

对固定的 `w,b`，最小可行松弛变量就是：

```text
margin_i = y_i * f(x_i)
xi_i = max(0, 1 - margin_i)
```

这个 `xi_i` 数值上等于逐样本hinge损失，但两个名字强调的角度不同：松弛变量来自约束，hinge损失来自无约束目标。

## 先看带符号间隔，不要只看分类对错

设 `margin = y * score`：

| 带符号间隔 | 样本位置 | hinge/slack |
| --- | --- | --- |
| `margin > 1` | 正确分类且在间隔外 | `0` |
| `margin = 1` | 正好位于间隔边界 | `0` |
| `0 < margin < 1` | 正确分类但进入间隔 | `1-margin` |
| `margin = 0` | 位于决策边界 | `1` |
| `margin < 0` | 已误分类 | 大于 `1` |

因此“预测正确”不等于“没有损失”。位于间隔内部的正确样本仍有hinge损失。

## C控制什么

```text
0.5 * ||w||^2          希望模型平坦、间隔宽
C * sum(hinge_i)       惩罚训练样本违反间隔
```

- `C` 小：更愿意容忍训练违例，通常得到更小的 `||w||`；
- `C` 大：更重视减少训练违例，可能用更大的 `||w||` 换取更小的总松弛；
- `C` 不是“越大越好”，它是正则化与训练拟合之间的取舍。

本专题的演示会在同一份数据上训练多个 `C`，直接比较权重范数、总松弛、目标分量和样本状态。

## alpha位置与样本状态

KKT条件给出：

```text
alpha_i = 0      -> margin_i >= 1
0 < alpha_i < C  -> margin_i = 1
alpha_i = C      -> margin_i <= 1
```

所以：

- 非支持向量通常位于间隔外；
- 自由支持向量通常正好位于间隔边界；
- 有界支持向量可以在间隔边界、间隔内部或被误分类。

注意反向推断不总是唯一，例如退化问题中，`margin=1` 的样本也可能达到上界。

## 形状约定

```text
scores:   (n,)
y:        (n,)
margins:  (n,)
slacks:   (n,)
alphas:   (n,)
weights:  (d,)
```

本专题故意拒绝 `(n,1)` 与 `(n,)` 静默广播，避免产生 `(n,n)` 的错误结果。

## 学生任务

1. 不运行代码，先手算给定 `score` 和 `y` 的带符号间隔；
2. 判断每个样本属于间隔外、边界、间隔内还是误分类；
3. 写出hinge损失并求和；
4. 根据alpha与C判断非支持、自由支持和有界支持；
5. 逐项检查KKT关系；
6. 比较多个C时，分别观察 `0.5||w||^2` 与 `C*sum(xi)`，不要只看准确率；
7. 修改一个离边界很远的样本，预测它是否会改变支持向量集合。

参考实现和测试只提供建设证据，不代表学习者已经完成这些任务。
