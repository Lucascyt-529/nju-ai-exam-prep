# 支持向量机4：核函数与非线性分类

## 从点积到核函数

线性SVM的对偶目标只通过样本点积 `x_i^T x_j` 使用特征。若存在映射 `phi`，可以把点积替换为：

```text
K(x_i, x_j) = phi(x_i)^T phi(x_j)
```

训练和预测都直接计算 `K`，不必显式构造可能很高维的 `phi(x)`，这就是核技巧。

本专题实现三种核：

```text
linear:      K(x,z) = x^T z
polynomial:  K(x,z) = (gamma*x^T z + coef0)^degree
RBF:         K(x,z) = exp(-gamma*||x-z||^2)
```

## 形状主线

```text
X_train: (n,d)
X_query: (m,d)
K_train: (n,n)
K_query: (m,n)
alpha*y: (n,)
scores:  K_query @ (alpha*y) + b -> (m,)
```

最常见错误是把查询—训练核矩阵写反成 `(n,m)`，随后用转置勉强补救。请先写清楚“每一行对应一个待预测样本”。

## SMO哪里变、哪里不变

- 盒约束、等式约束、成对alpha更新和KKT条件不变；
- `eta`、误差和决策值都改为读取核矩阵；
- 非线性模型通常没有输入空间中的单一 `w` 可供恢复；
- 预测必须保存训练样本、alpha、标签、偏置和核参数。

只有 `alpha_i>0` 的训练样本真正参与预测，因此参考实现还提供只使用支持向量计算的决策函数，并验证它与全训练集表达式一致。

## XOR验证

XOR在二维输入空间中线性不可分。线性核不能完美拟合，而合适参数的RBF核可以得到四个正确标签。这个实验只证明核函数能表达该边界，不代表任意 `gamma` 和 `C` 都合适。

## 学生任务

1. 手算两个小矩阵的线性核；
2. 手算两个点的平方距离与RBF值；
3. 预测 `K(X,Z)` 的形状；
4. 验证训练Gram矩阵对称，RBF对角线为1；
5. 补全核化决策函数，解释为什么查询矩阵应为 `(m,n)`；
6. 比较XOR在线性核和RBF核下的预测；
7. 改变 `gamma`，观察“过宽”和“过窄”的核会怎样改变相似度。
