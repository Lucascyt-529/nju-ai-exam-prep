# 支持向量机6：表示定理与核线性判别分析

## 表示定理的迁移视角

许多带单调正则项的经验风险最优化问题，最优解可以写成训练样本核函数的有限线性组合：

```text
f(x) = sum_i alpha_i K(x, x_i) + b
```

这意味着即使隐式特征空间很高维，待学习的方向也无需在整个空间中任意搜索；训练数据张成的子空间已经足够。SVM、SVR、核岭回归和本专题的KLDA虽然损失与目标不同，却共享这种“查询—训练核矩阵乘系数”的预测形状。

表示定理不是说所有核方法都使用同一个优化问题，也不是说系数一定稀疏。SVM常因对偶约束得到许多零系数；KLDA通常会使用较多训练样本。

## 从LDA到KLDA

普通LDA寻找类间均值差大、类内散度小的投影方向。KLDA先把样本映射到隐式特征空间，再令投影方向写成：

```text
w = sum_i alpha_i phi(x_i)
```

于是训练样本投影为 `K_train @ alpha`，查询样本投影为 `K_query_train @ alpha`。参考实现构造核空间中的类间均值差向量和类内散度矩阵，用带正则的伪逆求解系数，并以两类投影均值中点作为阈值。

## 形状主线

```text
X_train:       (n,d)
X_query:       (m,d)
K_train:       (n,n)
K_query_train: (m,n)
alpha:         (n,)
projection:    (m,)
```

最常见错误是把查询核矩阵写成 `(n,m)`，然后靠转置补救。应先固定语义：每一行是一个待预测样本，每一列对应一个训练样本中心。

## 学生任务

1. 用线性核手算一个 `(m,n)` 查询核矩阵；
2. 从 `w=sum_i alpha_i phi(x_i)` 推导 `f(x)=k(x,X)^T alpha`；
3. 写出两类在核坐标中的均值差向量；
4. 验证核空间类内散度矩阵对称半正定；
5. 解释正则项为何能处理奇异散度矩阵；
6. 比较线性核与RBF核在XOR上的预测；
7. 说明KLDA系数不必像SVM支持向量系数那样稀疏。

参考实现通过不等于学习者已经独立掌握。
