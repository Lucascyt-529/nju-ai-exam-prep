# 支持向量机1：间隔几何与缩放

## 超平面和得分

二分类超平面写成：

```text
f(x) = w^T x + b
```

本专题使用 `y ∈ {-1,+1}`，得分大于等于0时预测 `+1`，否则预测 `-1`。

形状是：

```text
X:      (n,d)
w:      (d,)
b:      标量
scores: (n,)
y:      (n,)
```

## 函数间隔和几何间隔

第 `i` 个样本的函数间隔：

```text
y_i * (w^T x_i + b)
```

它为正表示分类正确，越大表示带符号得分越大。但若把 `w,b` 同时乘以正数 `c`，预测完全不变，函数间隔却也乘以 `c`。

几何间隔消除这项任意缩放：

```text
y_i * (w^T x_i + b) / ||w||
```

把 `w,b` 同时乘以正数时，分子和分母一起缩放，所以几何间隔不变。点到超平面的无符号距离则是：

```text
abs(w^T x_i + b) / ||w||
```

## 规范化和最小间隔点

若所有训练样本都被正确分开，可用最小函数间隔缩放参数，使：

```text
min_i y_i(w^T x_i+b) = 1
```

在这个规范化下，最小间隔点满足等号。对真正的最大间隔最优解，它们对应支持向量；本专题尚未求最优超平面，因此只把它们谨慎称为“当前超平面的最小间隔点”。

## 硬间隔目标

规范化约束固定缩放后，最大化几何间隔等价于最小化：

```text
0.5 * ||w||^2
```

下一批才会从这个约束优化问题进入拉格朗日对偶和SMO。本批不包含训练求解器。

## 学生任务

1. 手算四个二维点的得分与函数间隔；
2. 除以 `||w||` 得到几何间隔；
3. 把 `w,b` 同乘3，比较两种间隔；
4. 找出最小间隔点；
5. 把参数规范化到最小函数间隔为1；
6. 解释为什么仅把权重放大不能真的增大分类边界的几何间隔。
