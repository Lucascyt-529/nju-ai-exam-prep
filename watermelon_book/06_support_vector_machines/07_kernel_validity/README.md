# 支持向量机7：合法核函数、半正定与组合闭包

## 为什么“像相似度”还不够

核技巧要求存在某个特征映射`phi`，使：

```text
k(x,z) = phi(x).T @ phi(z)
```

对任意有限样本`X:(n,d)`，核矩阵`K:(n,n)`必须满足：

```text
K == K.T
c.T @ K @ c >= 0     对任意c:(n,)
```

数值上可用对称矩阵的最小特征值检查半正定；很小的负值可能只是浮点误差，所以必须写明容差。若候选核在某个数据集上出现明显负特征值，就已经构成反例；反过来，只在一个有限数据集上通过检查，并不能证明它对任意数据都合法。

## 五种候选函数

参考实现统一返回`K(X,Z):(m,n)`：

```text
linear:      x.T @ z
polynomial:  (gamma*x.T@z + coef0)^degree
RBF:         exp(-gamma*||x-z||^2)
Laplacian:   exp(-gamma*||x-z||)
Sigmoid:     tanh(gamma*x.T@z + coef0)
```

前四种在规定参数范围内是常用合法核。Sigmoid形式需要额外参数条件，不能仅凭`gamma>0`和`coef0<0`就对任意数据保证半正定；专题保留它作为“先验证，不盲信名称”的反例来源。

## 从Gram矩阵恢复有限特征坐标

若`K`对称半正定，可分解为：

```text
K = Q @ diag(lambda) @ Q.T
Phi = Q_positive @ diag(sqrt(lambda_positive))
K = Phi @ Phi.T
```

`Phi:(n,r)`只是当前有限样本的一组坐标，不等于恢复了核函数对整个输入空间的唯一显式映射。零特征值方向不提供距离或内积信息，可删去。

## 教材中的三种组合闭包

若`K1`和`K2`来自合法核，则下面仍是合法核矩阵：

```text
positive sum: gamma1*K1 + gamma2*K2, gamma1,gamma2>0
product:      K1 * K2                    逐元素乘法
scaling:      diag(g) @ K1 @ diag(g)
```

第三种写成函数就是`g(x) * k1(x,z) * g(z)`。`g`可以取负值或零，因为结果仍是合同变换；它不是把核矩阵每个元素随意乘一个不相关系数。

## 形状和数值边界

- 查询核矩阵`K(X,Z)`不要求方阵，只有训练Gram矩阵`K(X,X)`谈对称半正定；
- `np.linalg.eigvalsh`只适用于实对称矩阵，先检查对称误差；
- 核组合必须使用同一批、同一顺序的样本，因此各Gram矩阵形状必须完全相同；
- 正权和不能偷偷使用负权重；负权重可把PSD矩阵变成非PSD；
- 逐元素乘法不是普通矩阵乘法；这里依赖Schur乘积性质；
- 对称但不半正定仍不是合法Gram矩阵。

## 学习任务

1. 手算两个点的线性、RBF和拉普拉斯核值，并先预测矩阵形状；
2. 对`K=[[1,2],[2,1]]`找一个使`c.T@K@c<0`的`c`；
3. 用特征分解把一个秩为1的PSD矩阵恢复成`Phi:(n,1)`；
4. 分别验证正权和、逐元素积、`diag(g)Kdiag(g)`仍为PSD；
5. 把正权和改成负权组合，构造最小反例；
6. 比较RBF与拉普拉斯核随点距离变化的速度；
7. 运行Sigmoid候选反例，解释“有限反例能否证伪”和“有限通过能否证明”；
8. 独立写一个函数，输入Gram矩阵并输出对称误差、最小特征值和判断结论。
