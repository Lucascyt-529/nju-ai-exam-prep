# 支持向量机2：线性对偶与确定性SMO

## 为什么转到对偶变量

线性软间隔SVM的对偶问题是：

```text
maximize  sum(alpha_i)
          - 0.5 * sum_i sum_j alpha_i alpha_j y_i y_j K(x_i,x_j)

subject to  0 <= alpha_i <= C
            sum(alpha_i * y_i) = 0
```

本专题先用线性核 `K(x_i,x_j)=x_i^T x_j`。学到 `alpha` 后：

```text
w = sum_i alpha_i y_i x_i
f(x) = sum_i alpha_i y_i K(x_i,x) + b
```

只有 `alpha_i > 0` 的训练样本参与预测，它们是支持向量。

## 为什么一次更新两个alpha

若只改变一个 `alpha_i`，通常会破坏等式约束 `sum(alpha_i*y_i)=0`。SMO每次选两个下标 `i,j`，让它们沿保持等式约束的方向一起变化。

根据 `y_i` 与 `y_j` 是否相同，可以推导 `alpha_j` 的允许区间 `[L,H]`。新值必须裁剪到该区间，再由等式约束恢复 `alpha_i`。

## 确定性候选选择

教学实现按样本下标顺序检查第一个变量，再按 `|E_i-E_j|` 从大到小尝试第二个变量，平局选择较早下标。若首个候选由于区间退化或更新太小而失败，会继续尝试后续候选。

当二阶项退化（例如重复且标签相反的样本）时，代码直接比较区间两个端点的完整对偶目标，而不是跳过更新。

## KKT检查

令带符号间隔 `m_i=y_i f(x_i)`：

```text
alpha_i = 0      -> m_i >= 1
0 < alpha_i < C  -> m_i = 1
alpha_i = C      -> m_i <= 1
```

测试同时检查：

- `0 <= alpha <= C`；
- `sum(alpha*y)` 接近0；
- KKT残差很小；
- 可分数据上的原始目标与对偶目标接近；
- 对偶历史不下降。

## C已经出现，但软间隔仍会单独讲

盒约束上界 `C` 已经使这个求解器能够处理不可完全分离数据。下一批会专门解释松弛变量、hinge损失、`C` 的权衡和边界/误分支持向量；本批重点是对偶和SMO机制。

## 当前边界

这是适合小数据和考场理解的确定性简化SMO，只支持二分类线性核。它没有工作集缓存、收缩启发式和大规模优化，也尚未扩展多项式核、RBF核或SVR。

## 学生任务

1. 手算4个一维样本的Gram矩阵；
2. 写出两个alpha的 `[L,H]`；
3. 检查一次更新前后 `sum(alpha*y)`；
4. 从alpha恢复 `w`；
5. 按三种alpha位置检查KKT条件；
6. 比较原始目标和对偶目标；
7. 用重复异标签样本理解为什么需要退化端点处理。
