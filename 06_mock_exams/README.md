# 模拟机试

模拟模式只提供题目、输入数据、输出要求、时间和评分，不提供函数名、starter 框架、代码分块或解题步骤。模拟前不要打开 `reference/`、`expected/` 或自动评分实现。

## 正式流程

1. 新建独立作答目录并准备计时器；
2. 只打开试卷和 `data/`；
3. 120分钟内完成程序和结果文件；
4. 时间到后立即停止修改并备份提交；
5. 再运行评分器、查看评分细则和参考输出；
6. 记录用时、得分、失分原因和下一次改进动作。

## 评分边界

- 自动评分只能检查规定的功能和输出，不等于完整代码审查；
- 参考提交跑满分只证明试卷可解，不是学习者成绩；
- 未计时独立作答，不在 `LEARNING_STATUS.md` 中记录模拟成绩；
- 模拟暴露的基础问题回到对应专题修正，再做一题同结构变式。

两套试卷当前均已有题面、数据、评分、参考解和自动验收，不需要为了目录重构重写题目。
