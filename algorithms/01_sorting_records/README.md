# 算法1：稳定归并排序完整程序

## 题目

读取 `n` 条学生记录，每条包含不含空格的编号和整数分数。按分数从高到低输出；分数相同的记录保持输入顺序。编号必须唯一。

输入：

```text
5
S003 80
S001 95
S004 80
S002 70
S005 95
```

输出：

```text
S001 95
S005 95
S003 80
S004 80
S002 70
```

## 为什么手写归并排序

Python 内置排序本身稳定，但本题用于掌握算法结构：

```text
拆成左右两半
→ 分别递归排序
→ 合并两个有序序列
```

合并时分数相同必须先取左半边元素，稳定性才成立。

## 复杂度

- 时间：`O(n log n)`；
- 额外空间：`O(n)`；
- 递归深度：`O(log n)`。

## 无 OJ 运行方式

标准输入：

```bash
python reference/solution.py < data/sample_input.txt
```

文件输入并写文件：

```bash
python reference/solution.py data/sample_input.txt output.txt
```

## 学生任务和迁移

1. 先只完成解析和输出；
2. 手算一次左右合并；
3. 构造三个同分记录验证原顺序；
4. 将规则改为升序；
5. 增加第二排序键前，解释它与“稳定保序”的区别；
6. 构造重复编号、错误行数和非整数分数，给出清晰错误。
