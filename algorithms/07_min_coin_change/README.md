# 算法7：一维 DP 最少硬币与组合恢复

## 题目

给定互不重复的正整数面额和目标金额，每种硬币可无限使用。输出最少硬币数和一种确定的组合；不可达时输出 `-1` 和 `NONE`。

## 状态与转移

```text
dp[a] = 凑出金额a所需的最少硬币数
dp[0] = 0
dp[a] = min(dp[a-coin] + 1)
```

同时保存 `chosen_coin[a]`。从目标金额不断减去所选硬币，就能恢复具体组合。

## 平局规则

面额升序遍历。候选数量更少时更新；数量相同时，选择更小的“当前最后一枚硬币”。恢复后将组合升序输出，因此结果可复现。

## 复杂度

面额数为 `k`、目标为 `A`：时间 `O(kA)`，空间 `O(A)`。

## 学生任务和迁移

1. 手算每个金额的dp和chosen_coin；
2. 测试金额0；
3. 测试不可达金额；
4. 构造两个最优组合并检查平局规则；
5. 将无限硬币改为每枚只能用一次，解释遍历方向变化；
6. 同时运行标准输入和文件输入模式。
