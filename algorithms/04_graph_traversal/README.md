# 算法4：无向图 BFS、DFS 与连通分量

## 题目

读取一个编号为 `0..n-1` 的简单无向图和起点，输出：

- 从起点开始的 BFS 顺序；
- 从起点开始的 DFS 顺序；
- 整张图的所有连通分量。

邻接点按编号升序访问，连通分量从最小未访问点开始，因此边输入顺序不会改变答案。

## 关键区别

- BFS 使用队列，适合无权最短层数；
- DFS 使用栈或递归，沿一条路径深入；
- 两者都应在“加入待处理结构”时标记访问，避免同一节点重复加入。

本题的迭代 DFS 将升序邻居逆序压栈，使弹栈访问仍为升序。

## 复杂度

建立并排序邻接表后，遍历时间 `O(n+m)`；邻接表空间 `O(n+m)`。

## 学生任务和迁移

1. 手画队列和栈的变化；
2. 打乱边输入顺序，验证结果不变；
3. 增加孤立点，检查连通分量；
4. 去掉入队标记，构造重复入队反例；
5. 比较递归DFS与逆序压栈DFS；
6. 同时运行标准输入和文件输入模式。
