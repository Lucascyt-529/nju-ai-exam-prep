# 拓扑排序：课程依赖

输入课程数 n、依赖数 m，随后每行 `course prerequisite`。若能完成所有课程，输出字典序最小的一种合法顺序；有环输出 `IMPOSSIBLE`。

```text
输入:
2 1
1 0
输出: 0 1
```

边从 prerequisite 指向 course。Kahn 算法把所有入度0节点放入小根堆，每取出一个就删除其出边；最终不足 n 个节点说明存在环。时间 `O((n+m) log n)`，空间 `O(n+m)`。

边界：无依赖、多种合法顺序、自环、独立课程。变式：只输出 `YES/NO` 判断能否完成。最终从空文件写出图构造、入度、队列循环、环判断与 `main()`。
