# 算法2：二分边界完整程序

## 题目

给定非降序整数数组和若干查询。对每个目标 `x` 输出：

```text
x lower_bound upper_bound count
```

- `lower_bound`：第一个 `>= x` 的下标；
- `upper_bound`：第一个 `> x` 的下标；
- `count = upper_bound - lower_bound`。

下标范围允许等于 `n`，因此目标不存在时仍有明确插入位置。

输入：

```text
7 4
1 2 2 2 5 8 8
2 3 8 10
```

输出：

```text
2 1 4 3
3 4 4 0
8 5 7 2
10 7 7 0
```

## 循环不变量

使用半开区间 `[left, right)`：

- 初始 `[0,n)`；
- 循环条件 `left < right`；
- 中点 `middle = (left + right) // 2`；
- 结束时 `left == right`，就是边界位置。

## 复杂度

每个查询时间 `O(log n)`，额外空间 `O(1)`；总时间 `O(q log n)`。

## 学生任务和迁移

1. 手算目标等于重复段、位于两数之间、小于最小值和大于最大值；
2. 分别写 lower和upper，不先合并条件；
3. 用空数组验证返回0；
4. 故意把 `<`/`<=` 互换，找最小反例；
5. 检查输入数组未排序时应在解析阶段拒绝；
6. 同时运行标准输入和文件输入方式。
