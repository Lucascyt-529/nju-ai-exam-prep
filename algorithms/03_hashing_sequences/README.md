# 算法3：哈希计数、保序去重与两数配对

## 题目

读取整数序列和目标和，输出：

1. 按首次出现顺序排列的不同值；
2. 按首次出现顺序排列的 `值:频次`；
3. 第一对和为目标值的下标，若不存在输出 `NONE`。

输入：

```text
8 10
3 7 3 2 8 7 5 5
```

输出：

```text
unique: 3 7 2 8 5
frequencies: 3:2 7:2 2:1 8:1 5:2
pair: 0 1
```

## 确定性配对规则

从左向右扫描右端下标 `j`。若补数已经出现，立即返回它最早的下标 `i` 和当前 `j`。只有查找完成后才把当前值写入字典：

```python
complement = target - value
if complement in earliest_index:
    return earliest_index[complement], j
earliest_index.setdefault(value, j)
```

顺序不能颠倒，否则单个值可能错误地与自己配对。`setdefault` 保留最早下标，不被后续重复值覆盖。

## 复杂度

- 频次统计与保序去重：平均 `O(n)` 时间、`O(k)` 空间；
- 两数配对：平均 `O(n)` 时间、`O(k)` 空间；
- `k` 是不同值数量。
