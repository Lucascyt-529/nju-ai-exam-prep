# 算法6：Dijkstra 非负权最短路

## 题目

读取非负整数权重的简单有向图、起点和终点，输出最短距离和一条确定的最短路径。不可达时输出距离 `-1` 和 `NONE`。

## 最小堆与陈旧条目

发现更短距离时，将新 `(distance, node)` 压入堆。旧条目不会从堆中删除，因此弹出后必须检查：

```python
if popped_distance != distances[node]:
    stale_entries += 1
    continue
```

遗漏这一步通常不改变最终距离，但会重复处理节点，严重拖慢程序。

## 确定性平局规则

- 邻接表按 `(终点编号, 权重)` 排序；
- 堆天然按 `(距离, 节点编号)` 排序；
- 只有候选距离严格更小时才更新；
- 同长候选保留第一次发现的父节点。

因此边输入顺序不会改变输出路径。

## 适用边界

Dijkstra 要求权重非负。负权边在解析阶段直接拒绝；有负权时应使用其他算法，不能“试着跑一下”。零权边允许。

## 复杂度

邻接表和二叉堆实现为 `O((n+m) log n)` 时间、`O(n+m)` 空间。

## 学生任务和迁移

1. 手算距离数组和堆内容；
2. 构造先发现长路、后发现短路的陈旧条目；
3. 构造两条同长路径并解释确定答案；
4. 测试零权边、不可达和起终点相同；
5. 增加负权边，确认解析阶段拒绝；
6. 同时运行标准输入和文件输入模式。
