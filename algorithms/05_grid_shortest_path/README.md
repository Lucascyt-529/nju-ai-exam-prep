# 算法5：网格 BFS 最短路与路径恢复

## 题目

网格中 `.` 可通行、`#` 是障碍。给定起点和终点，输出最短步数与一条确定的最短路径。不可达时输出：

```text
distance: -1
path: NONE
```

## 为什么 BFS 能求最短路

每次移动代价都为1。BFS按距离层依次扩展，节点第一次入队时得到的距离就是最短距离。

方向顺序固定为：上、左、右、下。父节点在第一次入队时保存，终点找到后从终点反向追溯，再翻转得到路径。

## 复杂度

最多访问每个格子一次，时间和空间均为 `O(rows * cols)`。

## 学生任务和迁移

1. 手画队列、距离和父节点；
2. 验证起点等于终点时距离为0；
3. 用障碍封死终点；
4. 构造多条同长路径并解释方向顺序；
5. 将父节点保存时机从入队改到出队，观察重复状态；
6. 同时运行标准输入和文件输入模式。
