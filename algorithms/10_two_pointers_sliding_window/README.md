# 双指针与滑动窗口：最长无重复子串

输入一行字符串，输出最长无重复连续子串的长度，以及一个左闭右开区间 `left right`。并列时保留最早区间。

```text
输入: abba
输出:
2
0 2
```

窗口右端逐字符扩张；若字符已在当前窗口出现，左端跳到上次位置后一位。字典保存最后位置，每个下标最多进入和离开窗口一次，时间 `O(n)`、空间 `O(k)`。

边界：空串输出 `0` 和 `0 0`；全相同字符长度为1。变式：只输出区间对应的原字符串。完成 `starter.py` 后运行 `demo.py`、`check.py`，再从空文件写出含 `main()` 的程序。
