# 算法8：二维 DP 最长公共子序列

## 题目

读取两个不含空白字符的字符串，输出最长公共子序列长度和一个确定的LCS。空字符串用空行表示。

## 状态

```text
dp[i][j] = first[:i] 与 second[:j] 的LCS长度
```

若末尾字符相同，来自左上角加1；否则取上方和左方较大值。

## 回溯平局规则

从右下角回溯：

- 字符相同：记录字符并向左上；
- 上方更大：向上；
- 左方更大：向左；
- 两者相同：固定向上。

LCS可能不唯一，因此没有平局规则就无法保证输出一致。

## 复杂度

两个字符串长度为 `n,m`：时间和空间均为 `O(nm)`。

## 学生任务和迁移

1. 手填小型二维表；
2. 从右下角按父状态恢复序列；
3. 测试一个或两个空字符串；
4. 构造多个LCS并观察向上平局规则；
5. 只求长度时压缩到一维，解释为何路径恢复更困难；
6. 同时运行标准输入和文件输入模式。
