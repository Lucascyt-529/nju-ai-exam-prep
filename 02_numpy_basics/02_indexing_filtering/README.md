# NumPy 2：索引、筛选与同步操作

## 核心问题

本专题不只是记住切片语法，而是确保你能回答：

- 一次索引后结果是一维还是二维；
- 条件产生的是数值还是布尔掩码；
- 筛选或打乱 `X` 时，`y` 应如何保持对应；
- 从完整数据表切出标签后，标签应为 `(n,)` 还是 `(n, 1)`。

## 运行前预测

阅读 `guided_demo.py` 后先写下：

1. `X[[2, 0]]` 的形状和行顺序；
2. `X[:, 1] >= 20` 的值和形状；
3. 使用该布尔掩码筛选 `X` 后保留哪些样本；
4. 为什么不能分别随机打乱 `X` 和 `y`。

然后运行：

```bash
python 02_numpy_basics/02_indexing_filtering/guided_demo.py
```

## 迁移题

1. 筛选第0列位于闭区间 `[2, 4]` 的样本；
2. 将目标列从最后一列改为第一列；
3. 同时选择第0列和第2列，比较 `X[:, [0, 2]]` 与两次单列选择；
4. 固定同一个随机种子运行两次，解释结果为什么完全一致；
5. 故意分别打乱 `X` 和 `y`，构造一个能直观看出样本标签错位的最小反例。
