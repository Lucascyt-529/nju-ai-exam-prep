# 形状练习：reshape、转置与拼接

这一小节把“看懂形状”推进到“有目的地改变形状”。

## 三条规则

1. `reshape` 只能重新排列已有元素，前后元素总数必须相同；
2. `X.T` 交换二维数组的两个轴，`(n, d)` 变成 `(d, n)`；
3. 沿 `axis=0` 拼接时列数必须相同，沿 `axis=1` 拼接时行数必须相同。

例如：

```python
y = np.array([1.0, 2.0, 3.0])
y_column = y.reshape(-1, 1)  # (3, 1)，-1 由元素总数自动推断
y_again = y_column.reshape(-1)  # (3,)
```
