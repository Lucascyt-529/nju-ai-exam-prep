# NumPy 1：数组、形状与 axis

## 先建立统一语言

机器学习中的特征矩阵通常写成：

```text
X.shape == (n_samples, n_features)
```

- 一行代表一个样本；
- 一列代表一个特征；
- `axis=0` 沿样本方向压缩，通常得到“每个特征一个结果”；
- `axis=1` 沿特征方向压缩，通常得到“每个样本一个结果”。

不要只背最后两句话。每次都在纸上画出行和列，再判断压缩哪个维度。
