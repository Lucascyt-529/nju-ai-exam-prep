# NumPy 基础

本模块从零复习 NumPy，不默认已经熟悉数组 API。目标不是记住函数名，而是能在运行前判断形状、轴和广播结果。

## 学习顺序

| 顺序 | 专题 | 状态 |
| --- | --- | --- |
| 1 | `00_array_creation_dtypes/`：数组创建、维度、形状与 dtype | 已建参考闭环，等待学生练习 |
| 2 | `01_arrays_shapes_axes/`：二维形状、行列与 axis | 已建参考闭环，正在诊断 |
| 3 | `01_arrays_shapes_axes/reshape_practice/`：一维/二维转换、转置与拼接 | 已建参考闭环，等待学生练习 |
| 4 | `02_indexing_filtering/`：索引、筛选、特征/标签切分与同步打乱 | 已建参考闭环，等待学生练习 |
| 5 | `03_broadcasting/`：特征向量、样本列向量与安全标准化 | 已建参考闭环，等待学生练习 |
| 6 | `04_matrix_multiplication/`：矩阵乘法、线性输出与 Gram 矩阵 | 已建参考闭环，等待学生练习 |

任何专题都按 `learning_method.md` 执行：先预测，再运行；先小步补全，再独立实现；最后完成变式题。
