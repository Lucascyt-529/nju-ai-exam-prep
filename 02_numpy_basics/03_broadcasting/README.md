# NumPy 3：广播与向量化

## 为什么广播危险

广播可以省掉循环，但某些形状错误不会报错，只会得到含义错误的结果。本专题重点区分：

- `X.shape == (n, d)`：`n`个样本、`d`个特征；
- `feature_values.shape == (d,)`：每个特征一个数；
- `sample_values.shape == (n, 1)`：每个样本一个数。
