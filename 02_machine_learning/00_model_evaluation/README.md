# 模型评估

模型评估回答的不是“模型有没有把训练数据记住”，而是“模型对没有参与拟合的数据是否仍然有效”。本入口先掌握回归指标，再把训练集、验证集和测试集的职责贯穿到后续每个算法。

## 基本认识

机器学习希望模型在未见样本上也表现良好，这种能力称为泛化能力。训练数据上的误差是经验误差，只能说明模型对现有样本的拟合程度；真正关心的泛化误差无法直接知道，只能通过合理的数据划分和评估方法估计。

因此，模型评估不仅是训练结束后计算一个分数，还承担模型选择的任务。同一种算法可以有不同超参数，不同算法也可能在训练集上表现相近，必须在没有参与拟合的数据上比较。评估结果还会受数据划分、类别比例和指标选择影响，所以后续需要学习留出法、交叉验证和针对不同任务的性能度量。

这一部分是所有算法的共同入口：先明确用什么数据、什么指标评价，才知道训练得到的模型是否真正有用。

## 三份数据各做什么

```text
训练集：拟合模型参数
验证集：比较算法、超参数和预处理方案
测试集：方案完全确定后，只做最后一次评估
```

测试集不能反复用来挑模型，否则它实际上已经变成验证集，最终分数会偏乐观。选定方案后，是否合并训练集与验证集重训，要按题目要求明确说明。

当前假定 `y_true` 和 `y_pred` 是合法的一维 NumPy 数组，只练指标含义与模型比较，不在学生代码中加入 shape 或内容校验。

## 四个回归指标

令第 `i` 个样本的误差为：

```text
error_i = y_pred_i - y_true_i
```

| 指标 | 计算 | 直觉 | 越大还是越小越好 |
| --- | --- | --- | --- |
| MAE | `mean(abs(error))` | 平均偏离多少原始单位 | 越小越好 |
| MSE | `mean(error ** 2)` | 大误差受到更重惩罚 | 越小越好 |
| RMSE | `sqrt(MSE)` | 与目标值单位相同 | 越小越好 |
| R2 | `1 - SSE / SST` | 相比“永远预测均值”改善多少 | 越大越好 |

其中：

```text
SSE = sum((y_true - y_pred) ** 2)
SST = sum((y_true - mean(y_true)) ** 2)
```

`R2=1` 表示完美预测，`R2=0` 表示与预测真实值均值一样好，`R2<0` 表示还不如均值基线。它不是只能位于0到1之间。

## 函数职责

| 函数 | 输入 | 输出 | 下游用途 |
| --- | --- | --- | --- |
| `mean_absolute_error` | 真实值、预测值 | 一个浮点数 | 解释平均绝对偏差 |
| `mean_squared_error` | 真实值、预测值 | 一个浮点数 | 回归训练和模型比较 |
| `root_mean_squared_error` | 真实值、预测值 | 一个浮点数 | 用原目标单位解释误差 |
| `r2_score` | 真实值、预测值 | 一个浮点数 | 与均值基线比较 |

推荐实现顺序：

```text
mean_absolute_error
-> mean_squared_error
-> root_mean_squared_error
-> r2_score
```

RMSE 应直接复用 MSE；R2 要先区分模型残差平方和与真实值围绕均值的总平方和。

## 最小手算样例

```text
y_true = [1, 2, 3]
y_pred = [1, 4, 2]
error  = [0, 2, -1]
```

所以：

```text
MAE  = (0 + 2 + 1) / 3 = 1
MSE  = (0 + 4 + 1) / 3 = 5/3
RMSE = sqrt(5/3)

mean(y_true) = 2
SSE = 5
SST = (1-2)^2 + (2-2)^2 + (3-2)^2 = 2
R2 = 1 - 5/2 = -1.5
```

负 R2 在这里是有意义的：当前预测比 `[2, 2, 2]` 这个均值基线更差。

## 如何比较模型

不要只报告一个训练指标。一个最小的回归比较流程是：

1. 每个候选方案只用训练集拟合；
2. 在同一验证集上计算相同指标；
3. 根据任务重点选择主指标，例如强调大误差时优先 MSE/RMSE；
4. 方案确定后才在测试集上报告最终结果；
5. 同时给出简单基线，避免“指标看起来有数值但模型其实没用”。

分类模型需要混淆矩阵、precision、recall、F1、ROC/AUC 等指标，见 [模型评估扩展区](../../watermelon_book/02_model_evaluation_selection/README.md)。先在逻辑回归中理解它们的实际用途，再回来完成扩展函数。

## 常见错误

- MAE 忘记取绝对值，正负误差互相抵消；
- MSE 写成平方和，漏除样本数；
- RMSE 对每个误差单独开方，而不是对整体 MSE 开方；
- R2 的分母错误使用预测值的方差；
- 认为 R2 一定非负；
- 用训练集指标选择最终模型，或反复查看测试集；
- 比较两个模型时使用了不同的数据划分。
