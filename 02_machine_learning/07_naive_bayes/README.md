# 朴素贝叶斯

朴素贝叶斯是生成式分类器。它先估计每个类别出现的概率，以及在该类别下每个特征取值的概率，再用贝叶斯思想比较新样本属于各类别的可能性。

“朴素”来自条件独立假设：给定类别后，各个特征彼此独立。现实中这个假设常常不完全成立，但模型仍可能有效，而且训练与预测都很透明。

## 算法概览

贝叶斯决策从概率角度处理分类：如果能够知道样本属于各类别的后验概率，就可以按照后验概率或不同错误代价选择决策。朴素贝叶斯通过建立数据生成模型来估计这些概率。

直接估计所有特征的联合分布需要大量数据。条件独立假设把高维联合概率拆成多个单特征条件概率的乘积，大幅降低了参数数量和估计难度。这是一个很强的假设，却常能得到稳定而高效的分类器。

离散特征通常通过频数与平滑估计概率，连续特征可以假设类条件分布为高斯分布。朴素贝叶斯训练速度快、能自然输出概率，也是文本分类等高维任务中的常见基线。

## 分类公式

对类别 `c` 和样本 `x=(x_1,...,x_d)`：

```text
P(c | x) ∝ P(c) * Π_j P(x_j | c)
```

只需要比较类别大小时，分母 `P(x)` 对所有类别相同，可以省略。为避免许多小概率相乘下溢，实际计算对数：

```text
joint_log_score(c, x)
= log P(c) + sum_j log P(x_j | c)
```

选择分数最大的类别。若要概率，再用稳定的 softmax 归一化各类别对数分数。

## 离散特征与 Laplace 平滑

离散版本通过计数估计条件概率。如果某个“类别-特征值”组合在训练中没出现，直接频率估计会得到0，使整个联合概率变成0。

Laplace 平滑在每个计数上加 `alpha`：

```text
P(value | class)
= (count(value, class) + alpha)
  / (class_count + alpha * number_of_possible_values)
```

当前模型还为预测时未见的新取值保留未知桶，因此新值仍能得到有限分数。

## 连续特征的高斯版本

Gaussian NB 对每个类别、每个特征估计均值和方差，并假设：

```text
x_j | class=c ~ Normal(mean_cj, variance_cj)
```

预测时累加高斯对数密度。方差为0会导致除0和对数问题，所以使用 `variance_floor` 给方差设置下限。

## 函数职责

| 函数 | 输入 | 输出 | 下游用途 |
| --- | --- | --- | --- |
| `fit_categorical_nb` | 离散训练特征、标签、`alpha` | 先验与条件概率模型 | 离散预测 |
| `fit_gaussian_nb` | 数值训练特征、标签 | 类均值、方差和先验 | 连续预测 |
| `joint_log_scores` | 模型、新样本 | 每个样本对每类的对数分数 | 概率、标签 |
| `predict_proba` | 模型、新样本 | 类别概率 | 阈值与解释 |
| `predict` | 模型、新样本 | 最可能类别 | 分类指标 |

推荐先学离散主线：

```text
fit_categorical_nb
-> joint_log_scores
-> predict_proba
-> predict
```

再将拟合函数替换为 `fit_gaussian_nb`，观察后面的预测流程如何复用。

## 最小手算样例

```text
X = [[sunny], [sunny], [rainy], [rainy]]
y = [0, 0, 1, 1]
alpha = 1
```

先验：

```text
P(class=0) = 1/2
P(class=1) = 1/2
```

在类别0中，`sunny` 出现2次、`rainy` 出现0次。若已知取值有两种：

```text
P(sunny | class=0) = (2+1)/(2+2) = 3/4
P(rainy | class=0) = (0+1)/(2+2) = 1/4
```

类别1正好相反，所以 `sunny` 预测为0，`rainy` 预测为1。平滑让未出现组合仍保留非零概率。

## 如何评价

训练完成后使用验证集的 accuracy、precision、recall、F1 或混淆矩阵。概率是否校准是另一个问题：类别预测正确不代表概率值一定可靠。

文本或高维离散特征中，条件独立假设虽然强，但朴素贝叶斯常是有效基线。连续特征若明显不是单峰高斯分布，需要谨慎解释 Gaussian NB。

## 常见错误

- 计算的是 `P(class | feature)`，却当作 `P(feature | class)`；
- 多个小概率直接相乘导致数值下溢；
- 忘记类别先验；
- Laplace 分母没有同步增加全部可能取值的平滑量；
- 预测出现新离散值时直接查字典失败；
- Gaussian NB 对全部类别共用同一均值；
- 方差为0时没有下限；
- `predict_proba` 直接对大对数分数取指数，未先减去每行最大值。

## 自学闭环

1. 手算本页先验与四个条件概率；
2. 完成离散拟合和对数分数，再完成概率与标签；
3. 增加训练中未出现的组合，比较有无平滑；
4. 预测一个完全未见的离散取值，检查未知桶；
5. 用两类一维数值数据手算均值、方差并完成 Gaussian NB；
6. 比较两种模型的输入假设，而不是只比较函数名。

能从计数表独立写出对数联合分数，并解释为什么需要平滑和取对数，才算完成主线。
