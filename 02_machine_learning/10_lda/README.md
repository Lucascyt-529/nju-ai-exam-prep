# LDA

LDA（线性判别分析）是一种有监督线性投影方法。二分类时，它寻找一个方向，使同类样本投影后尽量集中、两类中心投影后尽量分开。

它与 PCA 不同：PCA 不看标签，只保留总体方差大的方向；LDA 使用类别标签，寻找有利于类别分离的方向。

## 算法概览

LDA 的基本思想是把高维样本投影到一条直线上，使同一类别的投影尽量靠近，不同类别的投影中心尽量远离。它不是单纯追求类别中心距离，而是用类内散布对这种距离进行约束。

二分类 LDA 最终仍得到一个线性决策边界，但它的出发点与逻辑回归不同：逻辑回归直接建立类别概率模型，LDA 则从投影后的类间与类内结构构造判别方向。

多分类时可以得到多个判别方向，但有效维数最多为类别数减一。LDA 因而既可以作为分类器，也可以作为利用标签信息的监督降维方法。

## 类均值与类内散度

二分类标签记为0和1，类别均值为：

```text
mean_0 = mean(X[y == 0], axis=0)
mean_1 = mean(X[y == 1], axis=0)
```

类内散度矩阵：

```text
Sw = sum_{x in class 0} (x-mean_0)(x-mean_0).T
   + sum_{x in class 1} (x-mean_1)(x-mean_1).T
```

它描述各类别内部沿不同方向的扩散程度。

## Fisher 投影方向

希望最大化：

```text
J(w) = (w.T @ (mean_1 - mean_0)) ** 2
       / (w.T @ Sw @ w)
```

一个最优方向与下面向量同方向：

```text
w = pinv(Sw + regularization * I) @ (mean_1 - mean_0)
```

使用伪逆可以处理重复特征造成的奇异矩阵；正则化让类内散度更稳定。`w` 乘任意正常数仍代表同一投影方向，因此不要把权重绝对尺度当作唯一答案。

## 投影与分类

```text
projection = X @ w
projected_mean_0 = mean_0 @ w
projected_mean_1 = mean_1 @ w
threshold = (projected_mean_0 + projected_mean_1) / 2
```

预测时根据投影位于阈值哪一侧选择类别。方向符号整体反转时，阈值和判断方向也必须同步反转。

## 函数职责

| 函数 | 输入 | 输出 | 下游用途 |
| --- | --- | --- | --- |
| `class_means` | 二分类数据与标签 | 两个类别均值 | 散度、方向、阈值 |
| `within_class_scatter` | 数据与标签 | `Sw` | Fisher 方向 |
| `fit_binary_lda` | 数据、标签、正则强度 | 权重与阈值 | 完整模型 |
| `project` | 数据、权重 | 一维投影 | 可视化与预测 |
| `predict_lda` | 数据、权重、阈值 | 类别标签 | 分类评价 |
| `fisher_ratio` | 数据、标签、方向 | 一个可分性比率 | 比较投影方向 |

推荐先做二分类：

```text
class_means
-> within_class_scatter
-> fit_binary_lda
-> project
-> predict_lda
-> fisher_ratio
```

多分类散度矩阵、最多 `K-1` 个方向和最近投影类中心预测属于第二层。

## 最小样例

```text
X = [[-2,0], [-1,0], [1,1], [2,1]]
y = [0, 0, 1, 1]
```

类别均值：

```text
mean_0 = [-1.5, 0]
mean_1 = [ 1.5, 1]
```

第一维能明显分开两类，第二维也有类别差异。LDA 会综合“均值差”和“类内散布”选择方向，而不是简单使用均值差本身。拟合后这4个训练样本应能被中点阈值正确分开。

## 如何评价与比较

- 分类任务使用验证集 accuracy、F1 等指标；
- `fisher_ratio` 只评价当前投影上的类间/类内关系，不替代泛化指标；
- 比较不同正则强度时只用验证集；
- 与 PCA 比较时使用相同下游模型和数据划分；
- 多分类 LDA 最多得到 `K-1` 个有效判别方向。

## 常见错误

- 用总体均值代替各类均值；
- 类内散度误写成类别均值之间的外积；
- 显式求 `Sw` 的普通逆，在奇异数据上失败；
- 投影时写成逐元素乘法；
- 权重方向反转后没有同步调整预测规则；
- 认为权重必须与参考答案逐元素完全相同；
- 把 LDA 当作无监督降维，与 PCA 混淆；
- 使用测试集选择正则强度。

## 自学闭环

1. 手算本页两个类别均值；
2. 用两个一维类别手算 `Sw`、方向和阈值；
3. 完成六个二分类函数并运行 `check.py`；
4. 将全部样本整体平移，验证预测不变；
5. 复制一个特征制造奇异散度，比较伪逆和正则化；
6. 比较同一数据上的 PCA 第一方向与 LDA 方向；
7. 最后再学习多分类的 `K-1` 维限制。

能从“类间分开、类内聚拢”推回 Fisher 比率，并区分 LDA 与 PCA，才算完成主线。
