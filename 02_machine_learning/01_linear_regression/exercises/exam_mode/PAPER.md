# 线性回归完整程序（建议 45 分钟）

读取 `data/train.csv`、`validation.csv`、`test.csv`。前三列依次为 `id,x,target`，测试文件只有 `id,x`。只用训练集拟合带截距线性模型，在验证集计算 MSE；保存模型为 `model.npz`，重新加载后预测测试集并保存 `predictions.csv`。

输出文件要求：预测表表头为 `id,prediction`，数值保留6位小数；终端打印验证 MSE。程序必须能从命令行直接运行。

评分：读取与列处理20分，训练25分，验证15分，模型保存/恢复15分，预测文件15分，程序入口与可运行性10分。
