# 变式迁移

至少独立完成一道：

1. **双特征**：构造 `y = 2*x1 - x2 + 3`，观察模型能否学到两个权重；
2. **带噪声**：给标签加入固定小噪声，比较梯度下降与最小二乘的参数和 MSE；
3. **秩亏特征**：令第二列恒等于第一列的 2 倍，解释参数不唯一但预测仍可正确；
4. **无截距**：构造经过原点的数据，调用 `fit_intercept=False` 并检查 `b == 0.0`；
5. **学习率变化**：分别把学习率缩小和放大 10 倍，比较损失历史，不只观察最后一次损失。
6. **训练与验证不同**：在训练集上拟合精确直线，在验证标签中加入固定噪声；解释训练损失很低时验证误差为什么仍可能更高，以及为什么不能用测试集反复调参。

能够复现原样例但不能完成变式时，不记为真正掌握。
