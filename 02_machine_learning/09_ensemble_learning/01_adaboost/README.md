# AdaBoost

AdaBoost 串行训练多个弱分类器。每一轮提高错分样本的权重，让下一个分类器更关注此前难以分类的样本，最后按各分类器能力加权投票。

当前基础学习器是决策树桩：只选择一个特征、一个阈值和一个方向完成二分类。

## 算法概览

Boosting 的出发点是把若干表现略好于随机猜测的弱学习器提升为强学习器。AdaBoost 使用样本分布连接各轮训练：前一轮分错的样本在下一轮占据更大权重，使不同弱学习器逐步形成互补。

最终模型是一个加性模型，每个弱分类器都带有与自身准确性相关的权重。表现较好的分类器在最终决策中话语权更大，接近随机猜测的分类器贡献很小。

AdaBoost 能把简单树桩组合成复杂边界，但也可能持续关注噪声点和错误标签。学习时既要观察训练错误下降，也要用验证集判断增加轮数是否仍有泛化收益。

## 一轮训练做什么

初始时每个样本权重相同：

```text
sample_weight_i = 1 / n
```

第 `t` 轮：

1. 在当前样本权重下找到加权错误率最低的树桩；
2. 计算弱分类器权重；
3. 提高错分样本权重，降低正确样本权重；
4. 重新归一化样本权重。

若标签和预测均为 `-1/1`：

```text
error_t = sum(sample_weight_i * [prediction_i != y_i])
alpha_t = 0.5 * log((1 - error_t) / error_t)

new_weight_i
∝ sample_weight_i * exp(-alpha_t * y_i * prediction_i)
```

正确分类时 `y_i * prediction_i = 1`，权重乘 `exp(-alpha)`；错分时乘 `exp(alpha)`。

## 最终预测

每个弱分类器给出 `-1/1` 预测：

```text
score(x) = sum(alpha_t * stump_t(x))
prediction = 1 if score >= 0 else -1
```

错误率越低于0.5，`alpha` 越大。若弱分类器与随机猜测相同，`alpha` 接近0，对最终结果几乎没有贡献。

## 函数职责

| 函数 | 输入 | 输出 | 下游用途 |
| --- | --- | --- | --- |
| `fit_weighted_stump` | 数据、标签、样本权重 | 最佳树桩及加权错误 | 每轮弱学习 |
| `update_sample_weights` | 当前权重、标签、预测、`alpha` | 归一化新权重 | 下一轮训练 |
| `fit_adaboost` | 数据、迭代轮数 | 树桩和权重序列 | 完整强分类器 |
| `predict` | 集成模型、新数据 | `-1/1` 标签 | 分类评价 |

推荐顺序：

```text
fit_weighted_stump
-> update_sample_weights
-> fit_adaboost
-> predict
```

## 最小手算样例

假设4个样本初始权重均为0.25，某树桩只错第4个样本：

```text
error = 0.25
alpha = 0.5 * log(0.75 / 0.25)
      = 0.5 * log(3)
      ≈ 0.5493
```

更新前的比例因子：

```text
前三个正确样本：0.25 * exp(-0.5493)
第4个错分样本： 0.25 * exp( 0.5493)
```

归一化后，第4个样本将占总权重的一半，下一轮会更重视它。

## 如何评价

训练错误可能随轮数很快下降，但轮数仍应通过验证集选择。还应观察：

- 每轮弱分类器错误率是否低于0.5；
- 样本权重是否归一化且保持非负；
- 哪些样本持续获得高权重，它们可能是难例或噪声；
- 验证指标是否在继续增加分类器后停止改善。

## 常见错误

- 用普通错误率代替样本权重加权错误率；
- `alpha` 的分子分母写反；
- 错分样本权重反而降低；
- 更新后忘记归一化；
- 每轮都重新设为均匀权重；
- 最终预测使用普通多数投票，忽略 `alpha`；
- 标签使用0/1却直接套用 `-1/1` 指数更新；
- 弱分类器错误率为0或0.5时没有考虑数值边界。

## 自学闭环

1. 手算本页错误率、`alpha` 和新权重；
2. 完成加权树桩，确认权重改变会影响最佳划分；
3. 完成单轮更新并检查权重和为1；
4. 比较1轮与3轮的训练预测；
5. 找出每轮权重最大的样本并解释原因；
6. 用验证集比较不同 `n_estimators`，不能只看训练正确率。

能说明 AdaBoost 为什么必须串行训练，以及错分样本如何影响下一轮，才算完成主线。
