# 南京大学人工智能学院机试公开证据与训练取舍

本文件只记录能够回溯到公开页面的证据，用于调整训练优先级。个人经验贴不是官方承诺，年份之间也可能换题；因此它只能支持“更值得优先练什么”，不能预测下一次原题。

## 院系边界（检索时必须先核对）

本项目目标是南京大学**人工智能学院**。以下名称属于不同学院，其机试经验不能直接当作目标考试证据：

- 南京大学智能科学与技术学院（常简称“南大智科”，苏州校区）；
- 南京大学智能软件与工程学院；
- 南京大学软件学院；
- 南京大学计算机科学与技术系。

2025年公开检索中较容易找到的“两道Python、两道C++、没有机器学习代码题”来自智能科学与技术学院，不是人工智能学院，已明确排除。学习者补充其了解为人工智能学院历年均有机器学习内容；当前未找到可公开核验的2025人工智能学院题面来独立证明“每一年”，但也没有找到相反的同院证据。因此训练计划继续把机器学习代码置于主线，不因其他学院换题而降级。

南京大学人工智能学院2025年硕士研究生复试细则规定机试为现场上机、90分钟，涉及“人工智能程序设计”，使用Python环境并要求掌握NumPy等基础工具包。该细则属于统考复试，不是本科生开放日/预推免原题，但能确认学院当前确实重视Python与NumPy程序能力。

### B级：明确参加年份和学院的个人回忆

一篇2024年（25届）南京大学人工智能学院参营回忆称，笔试与机试共120分钟；机试部分包括：

1. 读取训练CSV并输出样本数、属性数等指标；
2. 用朴素贝叶斯对训练数据分类；
3. 把预测标签输出到TXT；
4. 另有一问作者未记清；
5. 实现5折交叉验证。

该回忆同时强调时间紧，作者给机试预留40分钟后发现明显不够。页面没有提到脏数据、非法维度或为每个内部函数穷举异常类型。

来源：<https://avoid.overfit.cn/post/ce876c328b5144a9be64b62e4335c4dd>

另一篇较早的南大AI经历建议西瓜书前10章、数据结构和算法基础，但没有给出可核验的机试异常输入要求。

## 对训练方案的直接结论

### 最高优先级

- 在无OJ环境从文件读取到最终输出的完整程序；
- CSV/TXT基础读写、列切分、形状确认和严格输出路径；
- NumPy数组形状、索引、广播、统计量和矩阵计算；
- 朴素贝叶斯、kNN、K-means、线性模型等核心算法的可复写实现；
- 训练/预测、K折划分和结果保存的串联；
- Python基础算法题，以及题目范围变化时的语言适应能力；
- 在没有在线判题反馈时用小样例、形状打印和断言自检。

### 只在题目要求时重点处理

- 缺失值、非法数字、空行、错误表头等文件数据问题；
- 类别范围、概率范围、矩阵奇异等算法前置条件；
- 输入数据可能不合法时的错误码或异常信息。

### 低优先级工程扩展

- 为每个内部NumPy小函数重复编写完整`isinstance`检查；
- 穷举与题面无关的所有错误类型；
- 为错误信息设计复杂封装；
- 不影响判题结果的个人空格/引号风格。

基础阶段仍保留一个完整校验专题，目的是认识`ndim`、`size`、`isfinite`和错误形状；掌握后，限时练习不再重复堆砌这些模板。
