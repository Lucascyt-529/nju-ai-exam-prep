# 综合任务3：对数几率回归完整分类任务

## 完整流程

```text
读取 train/validation/test CSV
→ 只用训练集拟合均值和安全标准差
→ 变换三份数据
→ 只用训练集训练带L2的对数几率回归
→ 在验证集计算 accuracy/precision/recall/F1/AUC
→ 保存并重新加载预处理参数和模型
→ 输出测试概率与标签
```

核心程序只使用Python标准库和NumPy。

## 输入

训练与验证表头：

```text
sample_id,feature_1,feature_2,label
```

测试表头没有 `label`。标签只能为0或1，特征必须是有限浮点数。

## 输出

预测CSV：

```text
sample_id,probability,label
```

概率保留6位小数。指标文件固定顺序输出 `accuracy`、`precision`、`recall`、`f1`、`auc`。

模型 `.npz` 必须同时保存：训练均值、安全标准差、权重、截距和决策阈值。预测阶段重新加载这些参数，不能重新拟合测试集统计量。

## 默认训练参数

- learning rate：0.2；
- steps：1000；
- L2：0.05；
- threshold：0.5。

命令行允许修改它们，但迁移时必须在验证集比较，不能根据测试结果选择。

## 无OJ自测

- [ ] 标准化参数只来自训练集；
- [ ] 验证标签变化不会改变模型参数；
- [ ] 概率有限且位于 `[0,1]`；
- [ ] AUC使用概率而不是0/1标签；
- [ ] 单行测试仍保持二维特征；
- [ ] `.npz` 可用 `allow_pickle=False` 恢复；
- [ ] 预测和指标文件逐字节符合要求。
