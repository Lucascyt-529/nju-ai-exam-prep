# 综合任务4：K-means CSV无监督完整任务

## 题目目标

在没有OJ、测试集也没有真实标签的情况下，完成：

```text
读取训练/测试CSV
→ 只用训练集拟合缺失值填补均值和安全标准差
→ 对候选k分别执行多次K-means++初始化
→ 每个k保留簇内平方和最小的一次
→ 用训练集轮廓系数选择k（并列时选较小k）
→ 保存训练均值、标准差和标准化空间中的聚类中心
→ 重新加载模型并为测试样本分簇
→ 输出簇标签和训练诊断信息
```

核心程序仅使用Python标准库和NumPy。

## 输入输出

CSV表头固定为：

```text
sample_id,feature_1,feature_2
```

特征留空表示缺失，显式写`nan`或`inf`视为非法。输出`assignments.csv`包含`sample_id,cluster`，诊断文件按固定顺序记录`selected_k`、`silhouette`和`inertia`。模型文件必须能用`allow_pickle=False`恢复。

## 分层闯关

| 关卡 | 只实现什么 | 自测重点 |
| --- | --- | --- |
| 1 | CSV读取 | 表头、重复ID、空白转NaN、单行二维形状 |
| 2 | 预处理 | 参数只来自训练集、整列缺失、零方差安全处理 |
| 3 | 单次K-means | `(n,k)`距离、中心更新、空簇、收敛、标签规范化 |
| 4 | 选模 | 多次初始化取最小inertia、轮廓系数取最大、并列规则 |
| 5 | 保存恢复 | 均值/标准差/中心完整，测试阶段不重新拟合 |
| 6 | 完整程序 | 命令行、目录创建、严格输出和错误码 |

不要一开始复制完整参考答案。每过一关只运行对应测试；当你能解释这一关的数组形状和信息来源后，再进入下一关。

## 无OJ自检

- [ ] 每个训练样本恰好属于一个簇；
- [ ] 所有中心和距离都有限；
- [ ] 每个候选`k`进行了多次初始化；
- [ ] 同一`k`按inertia选运行，不跨`k`直接比较inertia；
- [ ] 不同`k`使用轮廓系数比较，且单点簇样本轮廓值按0处理；
- [ ] 测试缺失值使用训练均值；
- [ ] 保存前后预测完全一致；
- [ ] 聚类编号按中心字典序固定，输出可复现；
- [ ] 更换随机种子时最终结论稳定，而非只看一次幸运初始化。

## 迁移题

1. 把候选集合改为`2 3 4 5`，解释为什么不能允许`k=n`计算轮廓系数；
2. 加入第三个特征，列出所有形状的变化；
3. 把某一列设为常数，解释安全标准差为什么取1；
4. 将测试集整体平移，观察簇分配但不要重新拟合均值；
5. 输出每个测试样本到所属中心的距离，设计新的表头；
6. 用DBI替换轮廓系数，并说明指标方向由“越大越好”变成“越小越好”。
