# pandas 基础（后续辅助模块）

历史材料显示学习者过去接触过 Series、DataFrame、筛选、缺失值、统计和时间索引，但当前已经遗忘较多，不能视为掌握。

本模块将在 NumPy 基础练习完成后按以下顺序建设：

1. Series、DataFrame、索引和数据类型；
2. `read_csv`、`head`、`info` 和数据检查；
3. `loc`、`iloc`、列选择和布尔筛选；
4. 缺失值、重复值和类型转换；
5. 排序、分组统计、合并与简单透视；
6. pandas 与 NumPy 的转换和形状检查；
7. 用同一小任务对比 pandas 与纯 NumPy 实现。

pandas 是表格处理辅助能力，不代替 NumPy 机器学习算法。当前不加入依赖，等正式开始本模块时再确认环境并锁定版本。
