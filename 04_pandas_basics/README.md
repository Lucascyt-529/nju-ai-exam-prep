## 使用边界

- `Series` 通常是一列带索引的数据；`DataFrame` 是二维表格；
- `.loc` 按标签选择，`.iloc` 按整数位置选择；
- 转成 NumPy 前必须明确选了哪些列、目标 dtype 和期望形状；
- 不用 pandas 的模型 API 代替 NumPy 手写算法。
